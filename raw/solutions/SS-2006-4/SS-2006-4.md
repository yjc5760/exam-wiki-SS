## 3. 詳細解析

### (一) 板梁的主要特色及使用時機

**定義：**

板梁（Plate Girder）是以鋼板（翼板、腹板）透過銲接（或鉚接）組合而成的組合梁，有別於熱軋 H 型鋼（Rolled Section），板梁的斷面尺寸可依設計需求完全客製化。

**主要特色：**

| 特色 | 說明 |
|------|------|
| 斷面客製化 | 翼板、腹板尺寸可自由調整，針對彎矩/剪力需求最佳化配置材料 |
| 跨距大 | 腹板高度 $h$ 可遠大於熱軋型鋼，提供極大的 $I_x$，適合大跨距 |
| 自重效率高 | 材料集中於翼板（承彎矩），腹板薄（承剪力），截面材料效率高 |
| 腹板 $h/t_w$ 大 | 腹板纖細，可能發生剪力挫屈，需配置中間加勁板（Intermediate Stiffener） |
| 製造彈性 | 可配合變化彎矩沿梁長改變斷面（鱗片式翼板、腹板開孔等） |

**使用時機：**

1. **跨距超大**：橋樑、廠房吊車梁跨距達 30～60 m 以上，熱軋型鋼無法滿足
2. **載重特重**：需要極高彎矩強度，熱軋型鋼規格不足
3. **高度有限制**：在有梁深限制的情況下需要最大化斷面效率
4. **特殊形狀需求**：非對稱斷面（如混凝土板組合梁）、變截面梁
5. **工廠製作組合梁**：橋樑上部結構、大型廠房屋頂桁架

### (二) 板梁發生剪力挫屈的原因

**腹板受剪的應力狀態：**

腹板在純剪力作用下，板面上存在±45° 方向的主應力：
- **+45° 方向**：拉力主應力 $\sigma_1 = +\tau$（對角拉力）
- **−45° 方向**：壓力主應力 $\sigma_2 = -\tau$（對角壓力）

**挫屈機制：**

腹板的−45° 方向壓力主應力，使腹板如同一片受均勻壓力的薄板。當剪應力 $\tau$ 超過**臨界剪力挫屈應力** $\tau_{cr}$ 時，腹板沿對角壓力方向發生波浪形屈曲，即**剪力挫屈（Shear Buckling）**。

**臨界剪力挫屈應力（彈性）：**

$$\tau_{cr} = k_v \frac{\pi^2 E}{12(1-\nu^2)} \left(\frac{t_w}{h}\right)^2$$

其中 $k_v$ 為剪力挫屈係數（與加勁板間距 $a/h$ 有關）。

**為何板梁特別容易發生剪力挫屈？**

板梁腹板的 $h/t_w$（高厚比）遠大於熱軋型鋼（熱軋 H 型鋼 $h/t_w$ 通常 20～50，板梁可達 100～300）。$\tau_{cr}$ 與 $(t_w/h)^2$ 成正比，高厚比越大，$\tau_{cr}$ 越低，腹板越容易在達到材料剪力降伏強度 $0.6F_y$ 前就先發生剪力挫屈。

**示意圖（文字版）：**

```
┌─────────────────────────────────┐  ← 上翼板
│                                 │
│  腹板（薄，h/tw 大）            │
│     ↗  ↗  ↗  ↗  ↗             │  ← 對角拉力方向（+45°）
│  ↘  ↘  ↘  ↘  ↘               │  ← 對角壓力方向（−45°），引發挫屈
│                                 │
│  ⌒⌒⌒⌒⌒⌒（挫屈後波形）         │
│                                 │
└─────────────────────────────────┘  ← 下翼板
         ↑                ↑
       剪力 V             剪力 V
```

### (三) 避免板梁發生剪力挫屈的方法

**核心思路：** 提高臨界剪力挫屈應力 $\tau_{cr}$，使其超過設計剪力需求。

**方法一：加厚腹板（提高 $t_w$，降低 $h/t_w$）**

直接增加腹板厚度可提高 $\tau_{cr}$（與 $(t_w/h)^2$ 成正比），但代價是自重增加、材料效率降低，通常不是最佳策略。

**方法二：配置中間垂直加勁板（Intermediate Transverse Stiffener）**——最常用

在腹板上配置等間距的垂直加勁板（如角鋼、鋼板），將腹板分割為多個較小的板格（Panel），縮短板格的**縱橫比** $a/h$（加勁板間距/腹板高），顯著提高 $k_v$：

$$k_v = 5 + \frac{5}{(a/h)^2}$$

- $a/h$ 越小（加勁板越密）→ $k_v$ 越大 → $\tau_{cr}$ 越高
- 規範限制：中間加勁板間距 $a \leq 3h$（避免 $k_v$ 過低）

**方法三：配置縱向加勁板（Longitudinal Stiffener）**

沿腹板高度方向設置水平加勁板，提高腹板在彎曲壓力方向的挫屈強度，通常用於超大型橋梁板梁。

**規範控制（腹板高厚比限制）：**

$$\frac{h}{t_w} \leq \frac{14{,}000}{\sqrt{F_y(F_y + 70)}} \quad \text{（無加勁板）}$$

$$\frac{h}{t_w} \leq \frac{322}{\sqrt{F_y}} \quad \text{（有加勁板，$F_y$ 單位 kgf/cm²）}$$

### (四) 張力場效應（Tension Field Action）

**定義：**

腹板發生剪力挫屈後，並未立即喪失承載能力。腹板的剩餘強度來自於沿+45°對角方向（拉力主應力方向）發展出的**斜向拉力帶（Tension Field）**，由翼板與加勁板共同錨定這條拉力帶，形成類似**桁架**的傳力機制，此即**張力場效應（Tension Field Action）**。

**力學類比：**

挫屈後的腹板板格可類比為一個「N形桁架（Pratt Truss）」：
- 加勁板（垂直板）→ 壓力斜撐（Compression Strut）
- 腹板對角拉力帶 → 拉力斜桿（Tension Diagonal）
- 翼板 → 上下弦桿（承受水平分量）

**後挫屈剪力強度（含張力場效應）：**

AISC 規範中，含張力場效應的腹板標稱剪力強度（適用於有中間加勁板且無端板格限制的板格）：

$$V_n = 0.6 F_y A_w \left[ C_v + \frac{1 - C_v}{1.15\sqrt{1 + (a/h)^2}} \right]$$

其中：
- $C_v$：剪力挫屈折減係數（$C_v = 1.0$ 表示腹板未發生挫屈，$C_v < 1.0$ 表示已挫屈）
- 方括號內第一項 $C_v$：腹板已挫屈的殘餘剪力貢獻
- 方括號內第二項：**張力場效應**的後挫屈強度貢獻

**示意圖（文字版）：**

```
  加勁板       加勁板
     ‖            ‖
─────╫────────────╫─────  ← 上翼板（錨定拉力帶上端）
     ‖  ↗  ↗  ↗  ‖
     ‖↗  ↗  ↗  ↗ ‖       ← 對角拉力帶（+45°，張力場）
     ‖  ↗  ↗  ↗  ‖
─────╫────────────╫─────  ← 下翼板（錨定拉力帶下端）
     ‖            ‖
  加勁板（受壓）  加勁板（受壓）

拉力帶（腹板）+ 壓力撐（加勁板）+ 弦桿（翼板）= 類桁架傳力
```

**張力場效應的限制條件：**

規範規定以下情況**不得**考慮張力場效應：
1. 端板格（End Panel，梁端支承附近的第一個板格）——翼板無法提供足夠錨定力
2. $2A_w / (A_{ft} + A_{fc}) > 2.5$（腹板面積過大相對於翼板）
3. $h/b_f > 3$ 或 $h/b_f > 6$（翼板過窄，無法錨定拉力帶）

## 4. 關鍵觀念整理（考前速記）

| 考點 | 核心答案 |
|------|---------|
| 板梁使用時機 | 大跨距、大載重；熱軋型鋼規格不足時 |
| 剪力挫屈原因 | $h/t_w$ 大 → $\tau_{cr}$ 低；腹板在$-45°$壓力主應力下挫屈 |
| 防止剪力挫屈 | **配置中間垂直加勁板**（縮小板格，提高 $k_v$） |

## 5. 涉及陷阱

1. **剪力挫屈方向**：挫屈是沿**壓力主應力（−45°）** 方向發生，但張力場是沿**拉力主應力（+45°）** 方向發展，兩者方向正好垂直，考生常混淆。
2. **端板格不可考慮張力場**：端板格因翼板錨定力不足，規範規定不得利用張力場效應，必須以純剪力挫屈強度（$V_n = 0.6F_yA_wC_v$）設計。
3. **加勁板是「結果」不是「原因」**：加勁板是**防止**剪力挫屈的手段，並不是剪力挫屈發生的原因。
4. **$C_v = 1.0$ 不代表沒有挫屈疑慮**：$C_v = 1.0$ 僅代表腹板達到剪力降伏前不會挫屈，不代表 $h/t_w$ 一定小。

## 6. 進階探討

**板梁 vs 熱軋型鋼的設計哲學差異：**

熱軋型鋼的腹板高厚比小（$h/t_w \leq 2.45\sqrt{E/F_y}$），幾乎不發生剪力挫屈，剪力強度以全腹板降伏 $V_n = 0.6F_yA_w$ 計算（$C_v = 1.0$）。板梁則主動接受腹板會挫屈的事實，利用張力場效應發揮後挫屈強度，以「犧牲腹板板件穩定性換取整體重量效率」為設計哲學。

**Wagner 張力場理論：**

張力場效應最早由 Wagner（1929）在航空結構中提出（Aircraft Skin Panels），後來被 Basler（1961）引入鋼結構板梁設計，是現代板梁設計規範的理論基礎。
