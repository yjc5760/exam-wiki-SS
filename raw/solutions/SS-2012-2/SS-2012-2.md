## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：** 韌性抗彎矩構架（Ductile Moment Resisting Frame，即 Special Moment Frame / SMF）的耐震哲學是以**大塑性變形消能**，要求梁在強地震下能進入塑性而不崩塌。本題考驗考生對台灣耐震規範中「容量設計（Capacity Design）」思想的理解——先確保塑鉸形成，再確保側向支撐讓塑鉸有足夠轉動能力。

**出題者測驗的能力：**
1. 韌性構架的塑性轉角量化規定（數值記憶 + 概念理解）
2. 側向支撐在塑鉸區的功能（防止 LTB 發生在非彈性轉動阶段）
3. 側撐設計原則與施作圖示（設置位置、間距計算）

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 問（一）：塑性轉角需求規定（7 分）

#### 背景概念

韌性抗彎矩構架（SMF）的設計目標是在設計地震下，梁柱接頭能承受大變形而維持完整承載力。規範要求接頭須通過資格測試，驗證其在循環載重下的塑性轉角能力。

#### 規範規定（依台灣耐震設計規範 / AISC 341）

**韌性抗彎矩構架（Special Moment Frame，SMF）接頭的塑性轉角需求：**

$$\theta_p \geq 0.04 \text{ rad}$$

即接頭必須能夠承受**至少 0.04 弧度（4%）的塑性層間位移角**，並在此轉角下維持不低於預期塑性彎矩 $M_p$ 的 80% 強度。

**規範要求內容：**

1. **最低塑性轉角要求：** $\theta_p = 0.04$ rad（弧度）

2. **強度保持要求：** 在塑性轉角達 0.04 rad 時，接頭之標稱彎矩強度不得低於 $0.8M_{pe}$，其中 $M_{pe} = R_y F_y Z_x$（預期塑性彎矩，考慮材料超強因子 $R_y$）

3. **資格測試方式：** 接頭須通過試驗驗證（以試驗結果作為依據）或依規範認可的預先資格接頭設計（Prequalified Connections）

4. **台灣規範補充：** 台灣建築物耐震設計規範對「韌性抗彎矩構架」的層間位移角容許值 $\Delta_a = 0.025$ hs（hs 為層高），其中 $\theta_p$ 為非彈性部分，約對應結構的非線性變形能力。

#### 小結

$$\boxed{\theta_p \geq 0.04 \text{ rad（SMF 韌性抗彎矩構架最低要求）}}$$

### 問（二）：梁塑鉸區側向支撐之目的、設計原則與施作（18 分）

#### (A) 設置側向支撐之目的

在強烈地震下，韌性構架之梁端將形成塑性鉸（Plastic Hinge）。進入大塑性轉角後，梁翼板壓縮側的抵抗能力若受到**側向扭轉挫屈（LTB）**的干擾，將使彎矩強度驟降，導致構架提前崩塌。

側向支撐的目的在於：
1. **防止非彈性 LTB：** 在梁端塑鉸形成後仍能保持彎矩強度，完成能量消散
2. **確保穩定的遲滯迴圈：** 讓結構在反復地震中維持飽滿的遲滯面積（消能能力）
3. **防止全面扭轉失穩：** 使梁在塑性轉動過程中不發生平面外位移與扭轉

#### (B) 設計原則

**側向支撐最大間距（依 AISC 341 / 台灣耐震規範）：**

在梁塑鉸區內，相鄰側向支撐間距 $L_b$ 須滿足：

$$L_b \leq \frac{0.086 r_y E}{F_y}$$

其中：
- $r_y$：梁弱軸迴旋半徑（mm）
- $E$：彈性模數（MPa）
- $F_y$：鋼材降伏強度（MPa）

**數值舉例（SM490 / SN490B 鋼材，$F_y = 325$ MPa，E = 200,000 MPa）：**
$$L_b \leq \frac{0.086 \times r_y \times 200{,}000}{325} = 52.9 r_y \approx 52 r_y$$

**實務上常用：**
$$L_b \leq 52 r_y \text{（對應耐震緊實斷面之側撐間距）}$$

**側向支撐之強度需求：**

每道側向支撐（含翼板上下兩側）之側向支撐力須不低於：

$$F_{brace} \geq 0.06 M_{pe} / h_o$$

其中 $h_o$ 為上下翼板中心間距離。

這保證在梁形成塑鉸且翼板出現大壓縮力時，側撐仍能抵抗翼板的側移傾向。

**側向支撐設置位置：**

1. **在預期塑鉸區附近（柱面外 $d$ 至 $2d$）** 各設一道側撐（$d$ = 梁深）
2. 接著每間距 $\leq L_b$ 設一道，直到超出塑鉸影響區

#### (C) 施作方式繪圖說明

```
平面圖（梁柱平面視圖）：

      柱                梁
  |─────────|───────────────────────────
  |         |  ←d→|←d→   L_b    L_b
  |    ┌────┤      ⓪      ①      ②  ···
  |    │    |  塑鉸 預期位置
  |    └────┤  
  |         |
  |─────────|───────────────────────────

  ⓪ = 塑鉸位置（柱面外約 d 處）
  ① ② = 側向支撐位置（間距 ≦ Lb = 0.086ryE/Fy）
```

**立面圖（梁橫截面側撐方式）：**

```
          ← 梁腹板方向 →
   ─────────────────────     ← 上翼板
         ↑
   ─ ─ ─支撐桿─ ─ ─ ─     ← 側向支撐（接往梁上翼板）
   ─────────────────────     ← 上翼板
         │ 腹板
   ─────────────────────     ← 下翼板
         ↓
   ─ ─ ─支撐桿─ ─ ─ ─     ← 側向支撐（接往梁下翼板）
   ─────────────────────
```

**施作說明：**

| 做法 | 說明 |
|------|------|
| **直接支撐（Direct Bracing）** | 在翼板處設置水平斜撐桿，連至相鄰剛性構件（樓板或柱梁接頭） |
| **剛性樓板支撐** | 若有混凝土版，確保剪力釘提供足夠側向剛度，翼板上下皆與版固接 |
| **斜撐配置** | 斜撐應盡量接近翼板面（上下翼板），避免僅支撐腹板中性軸 |

**特別注意：**
- 側撐**必須對梁的上下翼板分別設置**（尤其下翼板，地震時下翼板受壓，易發生 LTB）
- 在 RBS（犬骨式）削弱接頭中，側撐位置需考慮塑鉸偏移至削弱斷面處

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 5.1 普通韌性（IMF）vs. 韌性（SMF）的塑性轉角要求差異

| 構架類型 | 最低 $\theta_p$ 要求 |
|---------|---------------------|
| 韌性抗彎矩構架（SMF） | ≥ 0.04 rad |
| 普通韌性抗彎矩構架（IMF） | ≥ 0.02 rad |

### 5.2 為何梁端需要側撐而柱端不強調？

梁承受反向彎矩，地震往返時翼板交替受壓；柱斷面通常較肥厚且受軸壓，LTB 問題不如梁嚴峻。但 SMF 規範對強柱弱梁亦有相應規定，確保塑鉸在梁端形成。

### 5.3 RBS 犬骨接頭的側撐策略

犬骨式接頭（RBS）刻意將塑鉸移離柱面，塑鉸位置可預測。此時：
- 第一道側撐設在 RBS 弱化區末端外側
- 側撐設計需考慮 RBS 截面下的塑性彎矩

### 5.4 考場作答重點

- 問（一）：明確寫出 $\theta_p \geq 0.04$ rad + 強度保持條件
- 問（二）：三段式作答（目的 → 原則 → 圖示），圖示至少標出側撐位置、間距符號與塑鉸位置
