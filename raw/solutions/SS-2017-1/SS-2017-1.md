## 2. 考題核心精神與出題者意圖 (Core Concepts & Examiner's Intent)

**核心觀念：工程材料選用的多面向判斷**

此題考查考生能否從「銲接性」、「韌性」、「Z向性質」、「現場施工條件」四個維度綜合評估鋼種，而非單純依強度排序。跨海大橋代表：（1）海洋環境低溫、（2）全銲接細節複雜、（3）施工條件嚴苛（預熱不易）、（4）厚板接合有層狀撕裂風險。

**出題者測驗重點：**
- 碳含量低 → 銲接性好，不易氫裂
- 碳當量（Ceq）低 → 抗銲接裂縫性好
- CVN 試驗溫度低、能量高 → 低溫韌性好，抵抗脆裂
- **垂直軋延方向（Z向）CVN** → 防止層狀撕裂（Lamellar Tearing），全斷面銲接必要條件
- 預熱溫度低（甚至不需預熱） → 海上或冬季施工容易

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### 選擇：**鋼種 D**

### 理由一：不需預熱（No Preheat Required）→ 現場施工最有利

**說明：**
- 鋼種 D 的碳含量極低（C < 0.11%），鋼材本身不易因銲接熱循環而形成硬化組織或氫裂縫，因此**不需要預熱**。
- 跨海大橋施工環境在海上或橋面作業，預熱作業困難（需整面加熱鋼板至特定溫度並維持，在風吹環境下不易達成）。
- 相較之下：A 需預熱至 66°C（最嚴苛），B、C 需預熱至 21°C，現場管控負擔大。

$$\boxed{\text{D：不需預熱} \Rightarrow \text{現場可操作性最高，節省施工時間與成本}}$$

---

### 理由二：具垂直軋延方向（Z 向）CVN 測試 → 防止層狀撕裂

**說明：**
- 全斷面銲接（Complete Joint Penetration，CJP）接合中，銲縫收縮力方向垂直於鋼板面（即 Z 方向），此力可能引發**層狀撕裂（Lamellar Tearing）**。
- 層狀撕裂是鋼板因厚度方向延展性不足（因非金屬夾雜物沿軋延方向排列），在收縮應力下沿平行板面方向開裂的破壞模式。
- 鋼種 D 的 CVN 試驗採**垂直軋延方向取樣**，直接驗證 Z 方向韌性，其他三種鋼材均為軋延方向取樣（無法確認 Z 向性質）。

$$\boxed{\text{D：垂直軋延方向 CVN} \Rightarrow \text{防止全斷面銲接接合之層狀撕裂破壞}}$$

---

### 理由三：極優的低溫韌性（−25°C，200J）→ 海洋低溫環境安全

**說明：**
- 跨海大橋在冬季或夜間可能處於低溫環境，若鋼材韌脆轉換溫度（DBTT）高，低溫下可能發生脆性斷裂。
- 鋼種 D 在 −25°C 下仍保持 200J 的高衝擊值，遠優於 A（0°C, 27J）及 B（21°C, 34J）。
- 200J >> 27J（一般最低要求）：即便面對低溫衝擊載重或地震動力效應，仍有充裕韌性裕度。

$$\boxed{\text{D：} \text{CVN at} -25°C = 200\text{J} \Rightarrow \text{海洋低溫環境下最佳脆裂防護}}$$

---

### 理由四：極低碳含量（C < 0.11%）→ 銲接性最優，抗氫裂最強

**說明：**
- 碳含量是影響銲接性的主要因素。碳含量高 → 熱影響區（HAZ）硬化 → 延展性下降 → 氫致裂縫（Hydrogen-Induced Cracking）風險高。
- 鋼種 D（及 C）的 C < 0.11%，遠低於 A（0.18～0.20%）和 B（< 0.23%）。
- 全斷面銲接接合需施銲於大斷面、厚板，銲接熱輸入大，低碳含量對 HAZ 保護更關鍵。

$$\boxed{\text{D：C} < 0.11\% \Rightarrow \text{HAZ 韌性佳，無氫裂風險，最適合全斷面銲接}}$$

---

### 補充理由五（若需第五點）：強度最高（Fy > 500 MPa）→ 跨海長跨度結構輕量化有利

跨海大橋跨距長，使用高強度鋼可減少構件截面尺寸，降低自重（Dead Load），對長跨度橋梁的設計效益顯著。

## 5. 關鍵爭議點與進階探討 (Critical Issues & Advanced Discussion)

### 為何 C 不如 D？

| 比較項目 | C | D |
|---------|---|---|
| CVN 取樣方向 | 軋延方向（縱向）| **垂直軋延方向（Z向）** |
| 預熱溫度 | 21°C 需預熱 | **不需預熱** |
| 強度 | > 485 MPa | > 500 MPa（略高）|

對於全斷面銲接的跨海橋，Z 向性質和無需預熱是決定性的優勢，C 無法滿足。

### 碳當量（Ceq）說明

碳當量是評估銲接性的綜合指標：
$$C_{eq} = C + \frac{Mn}{6} + \frac{Cr+Mo+V}{5} + \frac{Ni+Cu}{15}$$

鋼種 A 有 Ceq < 0.36 的明確規定，但預熱溫度仍需 66°C（因 C% 本身較高，HAZ 硬化風險大）。D 雖無 Ceq 規定，但因 C < 0.11% 極低，實際 Ceq 相當低，且不需預熱即可驗證其銲接性。
