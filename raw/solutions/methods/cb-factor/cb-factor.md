## 核心原理

### 物理意義

**$C_b$（彎矩梯度修正因子，Moment Gradient Modification Factor）** 是用來量化「非均勻彎矩分布對 LTB（側向扭轉挫屈）強度提升效果」的修正係數。

#### 從第一原理理解

LTB 的臨界強度公式（彈性段）是針對**均勻彎矩**（uniform moment）推導的：
$$M_{cr,\text{均勻}} = \frac{\pi}{L_b} \sqrt{EI_y GJ + \left(\frac{\pi E}{L_b}\right)^2 I_y C_w}$$

在均勻彎矩下，整個無支撐段每一截面都受到相同的 $M$，是 LTB 的**最不利情況（Cb = 1.0）**。

而實際梁在真實載重下，彎矩圖往往是**非均勻**的（三角形、拋物線、反曲線等）。此時：
- 只有少數截面承受最大彎矩 $M_{max}$
- 兩側強度較大的段落能「拉住」中間較弱的截面，**提供額外的 LTB 束制**
- 相當於縮短了「有效挫屈長度」

$C_b > 1.0$ 意味著：此非均勻彎矩分布的 LTB 強度，**比同長度均勻彎矩提高了 $C_b$ 倍**。

#### 直覺比喻

想像一根梁的無支撐段：
- **均勻彎矩（$C_b = 1.0$）**：整段都「懸掛在刀鋒上」，任何位置都等可能發生 LTB
- **三角形彎矩（$C_b > 1.0$）**：只有一端承受最大彎矩，另一端接近零——彎矩小的那端「拉住」彎矩大的那端，阻止 LTB 發展

## 設計公式

### AISC 公式（LRFD 梁設計）

$$\boxed{C_b = \frac{12.5 M_{max}}{2.5 M_{max} + 3M_A + 4M_B + 3M_C} \leq 3.0}$$

**各符號定義：**

| 符號 | 定義 | 取值注意 |
|------|------|---------|
| $M_{max}$ | 無支撐段內最大彎矩絕對值 | 永遠取**絕對值** |
| $M_A$ | 無支撐段 $1/4$ 點的彎矩絕對值 | 永遠取**絕對值** |
| $M_B$ | 無支撐段 $1/2$ 點（中點）的彎矩絕對值 | 永遠取**絕對值** |
| $M_C$ | 無支撐段 $3/4$ 點的彎矩絕對值 | 永遠取**絕對值** |

> ⚠️ **所有彎矩值均取絕對值**，無論正負（正彎矩、負彎矩都要取絕對值代入）。

### 公式解讀

分母 $2.5M_{max} + 3M_A + 4M_B + 3M_C$ 是一個**等效均勻彎矩（Equivalent Uniform Moment）**：

- 加權平均：中點權重（4）> 四分之一點（3）> $M_{max}$（2.5）
- 中點 $M_B$ 取較大權重，反映 LTB 在中點附近最容易發生的物理事實
- 彎矩梯度越陡（非均勻程度越高）→ 分母越小 → $C_b$ 越大

### 在 LTB 設計中的應用

$C_b$ 修正三個 LTB 區間的強度：

$$M_n = C_b \times M_{n,C_b=1.0} \leq M_p$$

| 區間 | $C_b = 1.0$ 公式 | 加入 $C_b$ 後 |
|------|----------------|-------------|
| 塑性區（$L_b \leq L_p$） | $M_n = M_p$ | **不適用**，$C_b$ 不影響此區間 |
| 非彈性 LTB（$L_p < L_b \leq L_r$） | $M_n = M_p - (M_p - M_r)\dfrac{L_b - L_p}{L_r - L_p}$ | $M_n = C_b \left[M_p - (M_p - M_r)\dfrac{L_b - L_p}{L_r - L_p}\right] \leq M_p$ |
| 彈性 LTB（$L_b > L_r$） | $M_n = M_{cr}$ | $M_n = C_b M_{cr} \leq M_p$ |

> 💡 **關鍵上限：** 加入 $C_b$ 後，$M_n$ 不得超過 $M_p$（截面塑性彎矩為絕對上限）。

## 邊界條件 / 常見情況對照表

| 彎矩分布類型 | 圖形描述 | $C_b$ 值 | 備註 |
|------------|---------|---------|------|
| 均勻彎矩 | ──── 水平線 ──── | **1.0** | 最不利情況，LTB 規範基準 |
| 線性（三角形）| ▲ 一端最大，一端為零 | **1.67** | 如懸臂梁自由端無載重端 |
| 拋物線（均布載重）| ⌢ 中點最大 | **1.14** | 簡支梁均布載重最常見 |
| 對稱三角形（中點集中載重）| △ 中點最大 | **1.35** | 簡支梁跨中集中載重 |
| 反曲線（雙曲率）| S 形 | **≈ 2.3** | 樓架梁受側風力時常見，最有利 |
| 懸臂梁（自由端無支撐）| ─ 固端最大 | **1.0\*** | \*自由端無束制，直接使用 $C_b=1.0$ |

> ⚠️ **懸臂梁特殊規定：** 自由端無側向束制時，AISC 規定直接取 $C_b = 1.0$，不適用上述公式。

## 計算步驟流程

**Step 1：確認無支撐段長度 $L_b$**
- 找出梁側向支撐點位置（翼板繫條 / 側向支撐點 / 柱接合點）
- $L_b$ = 相鄰側向支撐點間距

**Step 2：找出彎矩圖**
- 繪製（或讀取題目給出的）彎矩圖
- 針對**最弱的無支撐段**（$M_{max}/M_n$ 最大的段）計算

**Step 3：取四個關鍵點彎矩（絕對值）**

```
支撐點A ────┬────────┬────────┬────────┬──── 支撐點B
           L_b/4   L_b/2   3L_b/4
           M_A     M_B     M_C     （M_max 在段內任意位置）
```

- $M_A$ = 距支撐點 A 的 $L_b/4$ 處彎矩絕對值
- $M_B$ = 距支撐點 A 的 $L_b/2$ 處（中點）彎矩絕對值
- $M_C$ = 距支撐點 A 的 $3L_b/4$ 處彎矩絕對值
- $M_{max}$ = 該無支撐段內**任意位置**的最大彎矩絕對值

**Step 4：代入公式**
$$C_b = \frac{12.5 M_{max}}{2.5 M_{max} + 3M_A + 4M_B + 3M_C}$$

**Step 5：確認上限**
$$C_b \leq 3.0$$（通常工程實務中 $C_b$ 鮮少超過 2.5）

**Step 6：計算修正後 $M_n$**
- 確認 $L_b$ 所在 LTB 區間
- 代入對應公式，乘以 $C_b$，並與 $M_p$ 取小值

## 推導示範

### 範例：簡支梁中點設有側向支撐，集中載重作用於梁端（三角形彎矩）

**題目設定：**
- 簡支梁總跨度 9m，梁中點設側向支撐（A, B 為端支撐，C 為跨中支撐）
- 分析 A-C 段（$L_b = 4.5$ m），此段彎矩為三角形（線性）
- A 端彎矩 = 0（簡支端），C 端彎矩 = $M_{max}$（最大值）

**實際算例（取 $M_{max}$ = 42 kN·m）：**

```
A端(M=0) ────┬────────┬────────┬────────┬──── C端(M=42)
             MA=10.5  MB=21   MC=31.5
```

線性彎矩分布：
- $M_A = M_{max} \times \frac{1}{4} = 42 \times 0.25 = 10.5$ kN·m
- $M_B = M_{max} \times \frac{2}{4} = 42 \times 0.50 = 21.0$ kN·m
- $M_C = M_{max} \times \frac{3}{4} = 42 \times 0.75 = 31.5$ kN·m

$$C_b = \frac{12.5 \times 42}{2.5 \times 42 + 3 \times 10.5 + 4 \times 21 + 3 \times 31.5}$$
$$= \frac{525}{105 + 31.5 + 84 + 94.5} = \frac{525}{315} = \mathbf{1.667}$$

> 💡 **驗證：** 線性（三角形）彎矩分布的理論 $C_b = 5/3 \approx 1.667$，完全吻合。

---

### 範例（來源：Mastering the Cb Factor, PDF p.7）

**題目設定：**
- 簡支梁，跨中有集中載重 P
- 側向支撐：兩端 + 梁中點（C 點）
- 分析右半段（C–B，$L_b = L/2$）
- C 點彎矩 = $M_{max} = PL/4$，B 點（右端簡支）彎矩 = 0
- 段內為**三角形彎矩**（從最大值線性降至零）

**計算（令 $M_{max}$ = 42 tf·m）：**
- $M_A$ = 42 × 3/4 = 31.5
- $M_B$ = 42 × 2/4 = 21.0（中點）
- $M_C$ = 42 × 1/4 = 10.5

（從 C 端量起：C 為支撐點，B 為右端簡支；彎矩從左到右：42→31.5→21→10.5→0）

$$C_b = \frac{12.5 \times 42}{2.5 \times 42 + 3 \times 31.5 + 4 \times 21 + 3 \times 10.5}$$
$$= \frac{525}{105 + 94.5 + 84 + 31.5} = \frac{525}{315} = \mathbf{1.667}$$

## 考場安全答法

**第一步（30 秒）：** 判斷是否需要算 $C_b$
- 題目給定 $C_b = 1.0$ → 直接使用，無需計算
- 題目說「均勻彎矩」→ $C_b = 1.0$
- 題目給出彎矩圖 → 必須算

**第二步（90 秒）：** 讀取四個點的彎矩

> ⚠️ 最常見陷阱：**忘記取絕對值**。反曲線（正負彎矩混合）時，MA、MB、MC 都取絕對值！

**第三步（30 秒）：** 代入公式，記住上限 $C_b \leq 3.0$

**第四步（30 秒）：** 將 $C_b$ 乘入 LTB 公式，記住上限 $M_n \leq M_p$

$$\boxed{C_b = \frac{12.5 M_{max}}{2.5 M_{max} + 3M_A + 4M_B + 3M_C} \leq 3.0 \quad \Rightarrow \quad M_n = C_b \times M_{n,1.0} \leq M_p}$$
