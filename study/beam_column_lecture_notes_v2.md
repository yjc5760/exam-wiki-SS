## 第一節：為什麼需要「梁柱桿件」獨立討論？

### 1.1 實際結構中的受力情形

在結構力學課程中，為簡化計算，我們習慣將柱定義為「純受軸壓」、梁定義為「純受彎矩」。然而，實際鋼結構框架中，**沒有任何構件能真正做到只受單一形式的力**：

| 構件 | 主要受力 | 兼受的其他力 | 成因 |
| :--- | :--- | :--- | :--- |
| 外框柱 | 軸壓 $P$ | 彎矩 $M$ | 側向風力、地震力、框架側移 |
| 偏心柱 | 軸壓 $P$ | 彎矩 $M$ | 梁與柱偏心接合、不對稱樓載重 |
| 中間梁 | 彎矩 $M$ | 軸力 $P$ | 斜撐系統、溫度效應、合成梁軸向力 |

同時承受**軸力 $P$** 與**彎矩 $M$** 的構件，統稱為**梁柱桿件（Beam-Columns）**，此為鋼結構設計中最具挑戰性、也最常出現在技師考試中的題型。

### 1.2 為何不能單獨各算、各自驗核？

直覺上，學生可能認為只要分別確認「軸壓強度 $\phi_c P_n > P_u$」且「彎矩強度 $\phi_b M_n > M_u$」，構件就安全了。這個想法**根本性地錯誤**，原因有二：

**❶ 強度的耦合折減**：斷面同時承受軸應力與彎曲應力，兩者共同佔用斷面的塑性承載容量，使彼此的可用強度都被壓縮。如同一個人同時用兩隻手各提一個行李箱，左手的行李重了，右手能提的重量就必然降低。

**❷ 二階效應的出現**：軸壓力作用在彎曲後的構件上，會在已有的彎矩之外額外產生「因變形而生的附加彎矩」——這就是**二階效應**，也是梁柱問題中最核心的力學特性。

## 第二節：二階效應的物理圖像

理解二階效應，是學習梁柱設計的第一道門檻。規範所有的放大係數，都源自這個物理現象。

### 2.1 P-δ 效應（構件層級，第一類二階效應）

想像一根兩端有彎矩 $M_1$、$M_2$ 的壓力柱，在端彎矩的作用下，桿身會產生撓曲，彎曲量為 $\delta$（**相對於直線連接兩端點的偏移量**）。

> **關鍵問題**：軸壓力 $P$ 此時落在哪裡？

答案是：它落在已偏離中心線的**彎曲後的截面形心**上！因此，$P$ 與彎曲位移 $\delta$ 會共同製造一個額外的「附加彎矩」$P \cdot \delta$，使得桿件中段實際受到的最大彎矩**大於**一階分析的端彎矩。

```
  一階分析（無變形）      二階效應（含 P-δ）
  
  M1 ←[─────────]→ M2     M1 ←[──╮──────╯──]→ M2
      |  P → →  |              | P → δ → P |
      軸力沿中心線             軸力沿彎曲後中心線
                               → 產生附加彎矩 P·δ
```

$P\text{-}\delta$ 效應的特徵：
- **發生尺度**：構件自身（局部效應）
- **放大工具**：係數 $B_1$（對應 LRFD），係數 $C_m/(1-f_a/F'_e)$（對應 ASD）
- **即使框架無側移，只要桿件有撓曲，就會發生**

---

### 2.2 P-Δ 效應（框架層級，第二類二階效應）

現在將視角拉高到整棟框架。在側向力（風力或地震力）作用下，框架各樓層產生側向位移 $\Delta$（稱為層間側移 Story Drift）。

這時，原本沿著各柱豎直傳遞的**重力載重 $P$（整層所有柱的總和）**，因為樓板整體偏移了 $\Delta$，實際上等效於有一個「翻轉力矩」$P \cdot \Delta$ 作用在框架上，進一步加大側移——這就是 $P\text{-}\Delta$ 效應。

```
  一階分析（側移框架）       加入 P-Δ 的二階效應
  
  ┌────────┐                 ┌────────┐ ← Δ
  │ 樓板  │← H               │ 樓板  │← H + P·Δ/L
  │       │                  │       │
  ╘═══════╛                 ╘═══════╛
  P↓  P↓  P↓               P↓  P↓  P↓  ← 同樣的重力P
                             但因側移Δ，產生額外翻轉傾向
```

$P\text{-}\Delta$ 效應的特徵：
- **發生尺度**：整個結構樓層（整體效應）
- **放大工具**：係數 $B_2$（對應 LRFD）
- **只有側移框架（Sway Frame）才需要計算**
- **靠桿（Leaning Column）的重力會加入 $\sum P_u$，顯著放大此效應**

---

### 2.3 兩種效應的比較總覽

| 比較項目 | $P\text{-}\delta$ 效應 | $P\text{-}\Delta$ 效應 |
| :--- | :--- | :--- |
| **發生尺度** | 構件本身撓曲 | 樓層側向位移 |
| **位移符號** | $\delta$（構件相對變形） | $\Delta$（樓層絕對位移） |
| **影響層面** | 個別構件彎矩增大 | 框架整體受力增大 |
| **LRFD 放大係數** | $B_1$ | $B_2$ |
| **計算彈性挫屈荷重時的 $K$** | 取 $K_1 = 1.0$ | 用對位圖查 $K_2$（Sway Frame） |
| **無側移框架** | 需計算 | 不需要（$B_2 = 1.0$） |
| **有側移框架** | 需計算 | 需計算 |

## 第三節：彎矩放大法（Moment Amplification Method）

規範以**彎矩放大法**（一種近似二階分析方法）處理上述效應。核心公式為：

$$\boxed{M_u = B_1 \cdot M_{nt} + B_2 \cdot M_{lt}}$$

### 3.1 名詞定義

**$M_{nt}$（Non-sway Moment，無側移彎矩）**  
在「假設框架不發生側移」的條件下，對結構進行一階分析所得的構件彎矩。典型來源：豎向重力載重、端部彎矩、對稱橫向載重。

**$M_{lt}$（Lateral Sway Moment，側移彎矩）**  
僅由**側向力**（或不對稱重力載重所引起的側移）產生的一階分析彎矩。典型來源：風力、地震水平力。

> **🎯 識別技巧**：先「鎖住」框架側移後做一階分析，得到的是 $M_{nt}$；再「解除」鎖定，只加水平力做分析，得到的是 $M_{lt}$。

---

### 3.2 B₁ 係數（處理 P-δ 效應）

$$B_1 = \frac{C_m}{1 - P_u / P_{e1}} \geq 1.0$$

**符號 $P_{e1}$（構件尤拉挫屈荷重）**：
$$P_{e1} = \frac{\pi^2 E I}{(K_1 L)^2}, \quad \text{其中 } K_1 = 1.0 \text{（不論框架有無側移，一律取 1.0）}$$

> [!WARNING]  
> **最常考的陷阱：$K_1 \ne K_{sway}$**  
> $B_1$ 在處理的是「假設兩端固定後，桿件本身的撓曲挫屈」，因此是無側移問題，$K_1$ 固定取 $1.0$。**絕對不可以把從對位圖查到的 $K$ 值（如 $K = 1.6$）帶進 $P_{e1}$！**

---

**符號 $C_m$（等效均勻彎矩係數）**：

$C_m$ 的物理意義：均勻彎矩分布對 $P\text{-}\delta$ 效應的放大程度最大；非均勻彎矩（尤其是雙曲率）對桿件中段的額外彎矩較小，故以 $C_m < 1.0$ 折減。

**無橫向載重、只有端彎矩的情況**（最常考）：

$$C_m = 0.6 - 0.4 \left(\frac{M_1}{M_2}\right)$$

其中 $|M_1| \le |M_2|$（$M_2$ 為絕對值較大的端彎矩）。

> [!IMPORTANT]  
> **$M_1 / M_2$ 符號約定——本節最重要的規則：**
>
> | 彎曲形態 | 圖形 | $M_1/M_2$ 帶入 | $C_m$ 結果 | 物理意涵 |
> | :--- | :---: | :---: | :---: | :--- |
> | **單曲率**（無反曲點） | `──╮──` | **正值** | $0.6 \sim 1.0$ | 桿身彎矩大，P-δ 放大效應明顯 |
> | **雙曲率**（有反曲點） | `─╮─╰─` | **負值** | $0.4 \sim 0.6$ | 桿身彎矩小，P-δ 放大效應輕微 |
>
> **記憶法**：「雙曲率最有利 → $C_m$ 最小（最有利的折減）→ $M_1/M_2$ 帶負號使公式自動給出小值」

**有橫向載重的情況**（保守簡化值）：

| 端部條件 | $C_m$ 取值 |
| :--- | :---: |
| 桿件兩端均有轉動束制（固接） | $0.85$ |
| 桿件至少一端為鉸接（可自由轉動） | $1.0$ |

---

### 3.3 B₂ 係數（處理 P-Δ 效應）

$$B_2 = \frac{1}{1 - \dfrac{\sum P_u}{\sum P_{e2}}} \geq 1.0$$

其中：
- $\sum P_u$：**同一樓層所有柱（含靠桿）**的設計軸壓力總和。
- $\sum P_{e2}$：同一樓層所有**主動參與抗側力**的柱之彈性挫屈荷重總和：
  $$P_{e2} = \frac{\pi^2 E I}{(K_2 L)^2}, \quad K_2 \text{ 由側移構架對位圖（Sway Nomograph）查得}$$

**替代公式**（利用層間勁度，無需計算各柱 $K$）：

$$B_2 = \frac{1}{1 - \dfrac{\sum P_u \cdot \Delta_{oh}}{\sum H \cdot L_c}} \geq 1.0$$

其中：
- $\Delta_{oh}$：該層在一階水平力 $\sum H$ 作用下的層間側移量（由結構分析取得）
- $L_c$：層高

> [!NOTE]  
> **靠桿效應（Leaning Column Effect）**：  
> 兩端鉸接的重力柱（靠桿）本身不能抵抗側向力，因此在計算 $\sum P_{e2}$ 時**不計入**（$P_{e2,靠桿} = 0$）。  
> 但靠桿上的重力**必須計入 $\sum P_u$**。  
> 這導致 $B_2$ 的分母變小、$B_2$ 值增大，使整個框架中的抗側力柱承受更大的設計彎矩。靠桿越重，框架越不穩定。

## 第四節：LRFD 梁柱互制公式

### 4.1 公式形式

規範以**分段線性折線**近似真實的 P-M 強度包絡面：

$$\text{若 } \frac{P_u}{\phi_c P_n} \geq 0.2：\quad \underbrace{\frac{P_u}{\phi_c P_n}}_{\text{軸力比}} + \frac{8}{9}\left(\underbrace{\frac{M_{ux}}{\phi_b M_{nx}} + \frac{M_{uy}}{\phi_b M_{ny}}}_{\text{彎矩比之和}}\right) \leq 1.0 \quad \cdots \text{(H1-1a)}$$

$$\text{若 } \frac{P_u}{\phi_c P_n} < 0.2：\quad \frac{P_u}{2\phi_c P_n} + \left(\frac{M_{ux}}{\phi_b M_{nx}} + \frac{M_{uy}}{\phi_b M_{ny}}\right) \leq 1.0 \quad \cdots \text{(H1-1b)}$$

| 強度項目 | 抗力係數 | 說明 |
| :--- | :---: | :--- |
| 軸壓強度 $\phi_c P_n$ | $\phi_c = 0.85$ | 含挫屈折減，依壓力桿件計算 |
| 彎矩強度 $\phi_b M_{nx}$ | $\phi_b = 0.90$ | 含 LTB 折減，依撓曲構材計算 |

### 4.2 P-M 互制圖的幾何意義

以無因次化座標繪製：橫軸 $= M / \phi_b M_n$，縱軸 $= P / \phi_c P_n$。

```
  P/φcPn
  1.0 ●
      |╲  H1-1a 區域（斜率較陡）
      |  ╲
  0.2 ●...●────────────────────── 折點
      |     ╲  H1-1b 區域（斜率較緩）
      |       ╲
  0.0 +────────●──── M/φbMn
     0        1.0

  折線定義：
  H1-1a：從 (0, 1.0) 到 (1/9, 0.2)（由 M=0, Pu/φcPn=0.2 求 M 的截距）
  H1-1b：從 (0, 0.2) 到 (1.0, 0)

  ▶ 設計點 (Mux/φbMnx, Pu/φcPn) 落在折線內側 → 安全 ✓
  ▶ 設計點落在折線外側 → 不足，需加大斷面 ✗
```

### 4.3 公式的物理直覺

**為何 H1-1a 有 8/9 係數？**  
當軸壓比很大（≥ 0.2）時，斷面大部分容量已被軸力佔用，彎矩的「殘餘空間」以更快的速率縮小（折線斜率更陡）。8/9 ≈ 0.889 是規範基於試驗結果的擬合係數。

**為何 H1-1b 分母是 $2\phi_c P_n$？**  
當軸壓比很小（< 0.2）時，軸力對斷面強度的直接削減效果減弱，規範用更平緩的折線段（H1-1b）來反映這個趨勢。

## 第五節：ASD 梁柱互制公式

ASD 使用**兩個控制方程式**，兩者均需滿足（取嚴者）：

**穩定性方程式（對應 H1-1a，含二階放大效應）**：
$$\frac{f_a}{F_a} + \frac{C_{mx} f_{bx}}{\left(1 - f_a/F'_{ex}\right)F_{bx}} + \frac{C_{my} f_{by}}{\left(1 - f_a/F'_{ey}\right)F_{by}} \leq 1.0$$

**斷面強度方程式（對應 H1-1b，桿端點截面不含放大效應）**：
$$\frac{f_a}{0.6 F_y} + \frac{f_{bx}}{F_{bx}} + \frac{f_{by}}{F_{by}} \leq 1.0$$

其中 ASD 的尤拉臨界應力（含安全係數 23/12 ≈ 1.92）：
$$F'_e = \frac{12\pi^2 E}{23(KL/r)^2}$$

**LRFD 與 ASD 的對應關係**：

| LRFD 符號 | ASD 符號 | 物理對應 |
| :--- | :--- | :--- |
| $P_u / \phi_c P_n$ | $f_a / F_a$ | 軸壓利用率 |
| $B_1$ | $C_m / (1 - f_a/F'_e)$ | P-δ 放大效應 |
| $M_{ux} / \phi_b M_{nx}$ | $f_{bx} / F_{bx}$ | 彎矩利用率 |

## 第六節：梁柱桿件設計驗核完整流程

```
┌─────────────────────────────────────────────────────────────────┐
│  STEP 1：計算設計軸壓強度 φcPn                                   │
│  ① 計算強弱軸細長比 (KxL/rx) 與 (KyL/ry)                       │
│  ② 取最大值 (KL/r)max 為控制細長比                              │
│  ③ 計算尤拉挫屈應力 Fe = π²E/(KL/r)²                           │
│  ④ 判斷挫屈模式：λ = 4.71√(E/Fy)                               │
│     KL/r ≤ λ → 非彈性挫屈：Fcr = 0.658^(Fy/Fe) × Fy           │
│     KL/r > λ → 彈性挫屈：Fcr = 0.877 Fe                        │
│  ⑤ φcPn = 0.85 × Fcr × Ag                                       │
├─────────────────────────────────────────────────────────────────┤
│  STEP 2：計算設計彎矩強度 φbMn                                   │
│  ① 確認斷面是否為結實斷面（檢核寬厚比）                          │
│  ② 確定無支撐長度 Lb，計算 Lp, Lr                               │
│  ③ 依 LTB 三區間判定 Mn                                         │
│  ④ φbMnx = 0.90 × Mn（雙軸彎矩需同時算弱軸 φbMny）             │
├─────────────────────────────────────────────────────────────────┤
│  STEP 3：計算二階設計彎矩 Mu = B1·Mnt + B2·Mlt                  │
│  ① 識別一階彎矩 Mnt（無側移）與 Mlt（有側移）                    │
│  ② 計算 Pe1 = π²EI/(L²)（K1 = 1.0，用彎矩作用方向的 I）       │
│  ③ 計算 Cm（由端彎矩比決定，注意雙曲率取負號）                   │
│  ④ B1 = Cm / (1 - Pu/Pe1)，若 < 1.0 取 1.0                    │
│  ⑤ 有側移框架：計算 ΣPe2（用對位圖 K2）或利用 Δoh 替代公式     │
│  ⑥ B2 = 1/(1 - ΣPu/ΣPe2)，若 < 1.0 取 1.0                    │
│  ⑦ Mu = B1·Mnt + B2·Mlt                                        │
├─────────────────────────────────────────────────────────────────┤
│  STEP 4：判斷適用公式                                             │
│  計算軸壓比 r = Pu / (φcPn)                                     │
│  r ≥ 0.2 → 使用 H1-1a                                          │
│  r < 0.2 → 使用 H1-1b                                          │
├─────────────────────────────────────────────────────────────────┤
│  STEP 5：代入互制公式，計算利用率（Demand-to-Capacity Ratio）    │
│  利用率 ≤ 1.0 → ✓ 安全                                         │
│  利用率 > 1.0 → ✗ 不足，需更換更大斷面                          │
└─────────────────────────────────────────────────────────────────┘
```

## 第七節：典型例題精析

### 例題 1：無側移框架，單軸彎矩（LRFD）

**題目條件**  
一無側移框架中的 H 型鋼柱，受以下 LRFD 設計載重：

- 鋼材：SN490B（$F_y = 3.3\ \text{tf/cm}^2$，$E = 2040\ \text{tf/cm}^2$）
- 斷面：假設已知斷面性質如下：
  - $A_g = 130\ \text{cm}^2$，$r_x = 14.0\ \text{cm}$，$r_y = 7.0\ \text{cm}$
  - 強軸斷面二次矩 $I_x = 25{,}480\ \text{cm}^4$
- 柱高 $L = 4.0\ \text{m}$；強弱軸兩方向均無側移（$K_x = K_y = 1.0$）
- 設計軸壓力：$P_u = 200\ \text{tf}$
- 一階端彎矩（強軸，單曲率）：$M_{nt, 大端} = +60\ \text{tf-m}$，$M_{nt, 小端} = +40\ \text{tf-m}$
- 無側移彎矩（$M_{lt} = 0$，無側移框架）
- **已知**強軸 LTB 計算後 $\phi_b M_{nx} = 80\ \text{tf-m}$（包含必要的 LTB 折減）

---

#### ▍ Step 1：計算設計軸壓強度 $\phi_c P_n$

控制細長比（弱軸控制）：
$$\left(\frac{KL}{r}\right)_{y} = \frac{1.0 \times 400}{7.0} = 57.1$$

尤拉挫屈應力：
$$F_e = \frac{\pi^2 \times 2040}{57.1^2} = \frac{20{,}132}{3{,}260} = 6.17\ \text{tf/cm}^2$$

交界細長比（判斷挫屈模式）：
$$\lambda = 4.71\sqrt{\frac{E}{F_y}} = 4.71\sqrt{\frac{2040}{3.3}} = 4.71 \times 24.87 = 117.2$$

因為 $57.1 < 117.2$，為**非彈性挫屈**：
$$F_{cr} = 0.658^{F_y/F_e} \cdot F_y = 0.658^{3.3/6.17} \cdot 3.3 = 0.658^{0.535} \cdot 3.3 = 0.812 \times 3.3 = 2.68\ \text{tf/cm}^2$$

設計軸壓強度：
$$\phi_c P_n = 0.85 \times 2.68 \times 130 = 295.9\ \text{tf}$$

---

#### ▍ Step 2：彎矩設計強度

已知 $\phi_b M_{nx} = 80\ \text{tf-m}$（題目給定）。

---

#### ▍ Step 3：計算二階設計彎矩 $M_{ux}$

**計算 $C_m$（單曲率彎曲，$M_1/M_2$ 取正值）**：
$$\frac{M_1}{M_2} = \frac{+40}{+60} = +0.667 \quad \text{（單曲率，正號）}$$
$$C_m = 0.6 - 0.4 \times 0.667 = 0.6 - 0.267 = 0.333$$

> **注意**：$C_m = 0.333$，表示單曲率彎矩分布相對均勻時，P-δ 放大效應較小（但 $B_1$ 還是要 ≥ 1.0）。

**計算強軸 $P_{e1}$（$K_1 = 1.0$，用 $I_x$）**：
$$P_{e1} = \frac{\pi^2 E I_x}{(K_1 L)^2} = \frac{\pi^2 \times 2040 \times 25480}{400^2} = \frac{512{,}660{,}784}{160{,}000} = 3{,}204\ \text{tf}$$

**計算 $B_1$**：
$$B_1 = \frac{C_m}{1 - P_u/P_{e1}} = \frac{0.333}{1 - 200/3204} = \frac{0.333}{1 - 0.0624} = \frac{0.333}{0.938} = 0.355$$

因 $B_1 = 0.355 < 1.0$，取 $B_1 = \mathbf{1.0}$。

**二階設計彎矩**：
$$M_{ux} = B_1 \cdot M_{nt} + B_2 \cdot M_{lt} = 1.0 \times 60.0 + 0 = 60.0\ \text{tf-m}$$

---

#### ▍ Step 4：判斷適用公式

$$\frac{P_u}{\phi_c P_n} = \frac{200}{295.9} = 0.676 \geq 0.2 \quad \Rightarrow \text{使用 H1-1a}$$

---

#### ▍ Step 5：代入互制公式

$$\frac{P_u}{\phi_c P_n} + \frac{8}{9}\left(\frac{M_{ux}}{\phi_b M_{nx}}\right) = 0.676 + \frac{8}{9} \times \frac{60.0}{80.0} = 0.676 + 0.889 \times 0.750 = 0.676 + 0.667 = 1.343$$

$$1.343 > 1.0 \quad \Rightarrow \text{斷面不足，需選用更大斷面。}$$

---

### 例題 2：有側移框架，雙軸彎矩（LRFD 含 B₂）

**題目條件**  
一有側移框架的角柱，承受以下 LRFD 設計荷重：

- 鋼材：SN490B（$F_y = 3.3\ \text{tf/cm}^2$）
- 設計軸壓力：$P_u = 100\ \text{tf}$
- 一階強軸端彎矩：$M_{nt,x} = 20\ \text{tf-m}$（無側移分量），$M_{lt,x} = 15\ \text{tf-m}$（側移分量）
- 弱軸：$M_{uy} = 5\ \text{tf-m}$（直接給定二階彎矩）
- 已查表或計算得：$\phi_c P_n = 300\ \text{tf}$、$\phi_b M_{nx} = 120\ \text{tf-m}$、$\phi_b M_{ny} = 50\ \text{tf-m}$
- 已知強軸 $B_1 = 1.0$（雙曲率，$C_m$ 小，$B_1$ 計算值 < 1.0）
- 已知 $B_2 = 1.20$（由 $\sum P_u / \sum P_{e2}$ 計算所得）

**計算強軸二階設計彎矩**：
$$M_{ux} = B_1 \cdot M_{nt,x} + B_2 \cdot M_{lt,x} = 1.0 \times 20 + 1.20 \times 15 = 20 + 18 = 38\ \text{tf-m}$$

**判斷適用公式**：
$$\frac{P_u}{\phi_c P_n} = \frac{100}{300} = 0.333 \geq 0.2 \quad \Rightarrow \text{H1-1a}$$

**代入互制公式**：
$$0.333 + \frac{8}{9}\left(\frac{38}{120} + \frac{5}{50}\right) = 0.333 + 0.889 \times (0.317 + 0.100) = 0.333 + 0.889 \times 0.417 = 0.333 + 0.371 = 0.704$$

$$0.704 \leq 1.0 \quad \Rightarrow \text{✓ 斷面安全，利用率 70.4\%。}$$

## 第八節：四大常見錯誤與澄清

### ❌ 錯誤 1：計算 $P_{e1}$ 時套入對位圖的 $K$ 值

**錯誤示範**：由對位圖查得有效長度係數 $K = 1.8$，直接代入計算 $P_{e1} = \pi^2 EI / (1.8L)^2$。

**正確做法**：計算 $B_1$ 用的 $P_{e1}$ **恆取 $K_1 = 1.0$**。對位圖的 $K$ 值（>1.0）是用來算 $B_2$ 裡面的 $P_{e2}$，兩者目的完全不同，不可混用。

---

### ❌ 錯誤 2：$C_m$ 公式的曲率正負號搞反

**錯誤示範**：題目給雙曲率彎矩（$M_A = +30$，$M_B = -40$），學生計算 $M_1/M_2 = 30/40 = 0.75$（正值），得 $C_m = 0.6 - 0.4 \times 0.75 = 0.3$，誤以為是雙曲率最有利的結果。

**正確做法**：雙曲率表示桿件有反曲點，兩端彎矩**方向相反**，$M_1/M_2$ 代入時取**負值** $= -0.75$，得 $C_m = 0.6 - 0.4 \times (-0.75) = 0.6 + 0.3 = 0.9$。  

> 弔詭之處：雙曲率雖然直覺上更穩定，但實際上 $M_1/M_2$ 的取值方式在不同版本規範定義中常有差異，務必確認所用規範的符號約定，並以「雙曲率時 $C_m$ 應更小、更有利」作為校驗依據。

---

### ❌ 錯誤 3：$\phi_c$ 與 $\phi_b$ 混用

**錯誤示範**：在計算 $\phi_c P_n$ 時誤用 $\phi_b = 0.9$，或在計算 $\phi_b M_n$ 時誤用 $\phi_c = 0.85$。

**正確做法**：

| 強度類型 | 抗力係數 | 記憶法 |
| :--- | :---: | :--- |
| 軸壓（Column） | $\phi_c = 0.85$ | **c** for **c**ompression，「壓」比「彎」折減更多（更不確定） |
| 受彎（Beam） | $\phi_b = 0.90$ | **b** for **b**ending |

---

### ❌ 錯誤 4：$M_u$ 忘記用二階彎矩

**錯誤示範**：題目給的一階分析彎矩 $M_{nt} = 40\ \text{tf-m}$，直接代入 H1-1a 作為 $M_{ux}$。

**正確做法**：互制公式中的 $M_{ux}$ 必須是**二階設計彎矩**，即 $M_{ux} = B_1 M_{nt} + B_2 M_{lt}$，即使無側移框架也需乘以 $B_1$（雖然 $B_1 \geq 1.0$ 有時等於 1.0，但不能跳過判斷步驟）。

---

## 第九節：課後討論與思考題

### 討論題 1：靠桿效應的敏感性分析

在一個單層框架中，包含 2 根剛接抗側力柱（各 $P_u = 50\ \text{tf}$）與 2 根鉸接靠桿（各 $P_u = 100\ \text{tf}$）。已知整層的彈性挫屈荷重總和（僅抗側力柱提供）$\sum P_{e2} = 600\ \text{tf}$。

**問**：如果靠桿的垂直載重增加為 $200\ \text{tf}$，$B_2$ 值的變化幅度為何？這對抗側力柱的設計有何實質影響？

*引導解答*：

- 原始 $\sum P_u = 2 \times 50 + 2 \times 100 = 300\ \text{tf}$，$B_2 = 1/(1-300/600) = 2.0$
- 修改後 $\sum P_u = 2 \times 50 + 2 \times 200 = 500\ \text{tf}$，$B_2 = 1/(1-500/600) = 6.0$
- $B_2$ 從 2.0 暴增為 6.0，側移彎矩放大 3 倍。在設計上，抗側力柱的 $M_{lt}$ 分量會急遽增大，原本安全的斷面可能完全不足。

> 這說明了靠桿效應在高軸壓、高側移框架中可能造成設計上的「爆炸式放大」，是實務設計中必須嚴密追蹤的效應。

---

### 討論題 2：直接二階分析 vs. 彎矩放大法

現代結構設計軟體大多具備「幾何非線性分析（P-Delta Analysis）」的功能，可以直接在矩陣分析中考慮幾何剛度矩陣的修正，得到含 $P\text{-}\delta$ 與 $P\text{-}\Delta$ 效應的「真二階彎矩」。

**問**：若已用軟體進行精確的二階分析，得到了構件的 $M$ 與 $P$，在代入 LRFD 互制公式時，是否還需要額外乘以 $B_1$ 和 $B_2$？

*引導解答*：

**不需要。** 精確的幾何非線性分析已在結構剛度矩陣中隱含地處理了所有二階效應，輸出的彎矩就是二階彎矩，可直接作為 $M_{ux}$ 代入互制公式。

$B_1$、$B_2$ 彎矩放大法是**一階分析搭配近似修正**的實用工具，其目的正是為了「模擬」二階分析的結果。若已做精確二階分析，再乘以 $B_1$、$B_2$ 就是**重複計算**，會使設計不必要地偏保守。
