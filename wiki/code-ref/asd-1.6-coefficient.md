# ASD 直接分析法中 1.6 倍載重的來源

**Code-Ref ID：** asd-1.6-coefficient
**規範來源：** AISC 360-10 Chapter C（直接分析法）
**相關題目：** [[SS-2011-1]]（直接考推導）

---

## 一、問題背景

直接分析法（Direct Analysis Method, DAM）是 AISC 2010 起引入的穩定分析方法，以**顯式二階分析**取代傳統有效長度法（ELM）中的 K 值。

DAM 基於 LRFD 框架開發：所有計算以因數化載重（$P_u = 1.2D + 1.6L$）為基準。

**問題：ASD 設計者使用的是服務載重（$P_a = D + L$），如何套用 DAM？**

---

## 二、推導

**LRFD 典型載重組合（重力主控）：**

$$P_u = 1.2 D + 1.6 L$$

**ASD 服務載重：**

$$P_a = D + L$$

**比值（兩者的倍率）：**

$$\frac{P_u}{P_a} = \frac{1.2D + 1.6L}{D + L}$$

分析兩個極端情況：

$$\text{若 } L = 0：\quad \frac{P_u}{P_a} = \frac{1.2D}{D} = 1.2$$

$$\text{若 } D = 0 \text{（}L \gg D\text{）：}\quad \frac{P_u}{P_a} = \frac{1.6L}{L} = 1.6$$

因此 $P_u/P_a$ 介於 **1.2 到 1.6** 之間。

**AISC 規範取最保守上限：**

$$\boxed{\alpha = 1.6 \quad \text{（ASD 進行直接分析法時）}}$$

---

## 三、1.6 倍的使用方式

1. ASD 設計時，將所有服務載重乘以 1.6：
   $$P_{analysis} = 1.6 \times P_a = 1.6(D + L)$$

2. 以此放大載重進行二階分析（計算 P-Δ、P-δ 效應、初始不完美）

3. 分析得到的內力除以 1.6，再與 ASD 容許強度比較：
   $$\frac{M_{2nd\text{-}order}}{1.6} \leq \frac{M_n}{\Omega_b}$$

**為何不用 1.2？** 因為 $L \gg D$ 時比值為 1.6，若只乘 1.2 會低估穩定效應；1.6 是確保任何載重比例下都保守的選擇。

---

## 四、與 ELM 的比較

| 比較項目 | 有效長度法（ELM） | 直接分析法（DAM） |
|---------|--------------|--------------|
| K 值 | 必須計算（對位圖或分析） | **可取 K = 1.0** |
| 剛度折減 | 無 | 需乘以 0.8τ_a |
| 二階分析 | 以 K 隱含近似 | 直接計算 P-Δ + P-δ |
| ASD 載重 | 服務載重直接用 | 服務載重 × **1.6** |
| 初始不完美 | 隱含於 K 值中 | 顯式模擬（等效假想水平力） |

**為何 DAM 可令 K=1.0？**
DAM 已在分析過程中考慮了所有穩定效應（二階效應 + 不完美 + 剛度折減），構件強度驗算不再需要 K 值補償，故 K = 1.0。

---

## 五、考試直接考點（SS-2011-1 第四小題）

> **題目：** 直接分析法中，ASD 設計者將服務載重乘以 1.6，請推導此係數 1.6 的來源。

**標準答法（3～5 行）：**
1. 說明 DAM 基於 LRFD 因數化載重開發
2. 寫出 $P_u/P_a = (1.2D+1.6L)/(D+L)$
3. 分析 $L \gg D$ 極限 → 比值趨近 1.6
4. AISC 取最保守上限 1.6 涵蓋所有載重比例
