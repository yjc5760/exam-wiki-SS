# Cb 彎矩梯度係數的來源

**Code-Ref ID：** cb-formula-derivation
**對應方法頁：** [[cb-factor]]（`wiki/methods/cb-factor.md`）
**對應概念頁：** [[BENDING-MODIFICATION-FACTOR-CB]]

---

## 一、為何需要 Cb？

LTB 公式（$M_{cr}$ 或 Lp/Lr 系統）是在**均勻彎矩（Pure Moment）** 假設下推導的。
均勻彎矩是「最不利」的彎矩分布——整個無支撐區間上彎矩大小相等，每一截面都承受相同的翻轉驅動力。

**實際梁**的彎矩幾乎不是均勻分布（有載重點、端部力矩梯度），彎矩梯度讓 LTB 強度比均勻彎矩情況**更高**。

$$C_b = \frac{M_{cr,\text{實際彎矩分布}}}{M_{cr,\text{均勻彎矩}}} \geq 1.0$$

$C_b$ 是一個**修正倍數**：均勻彎矩對應 $C_b = 1.0$（最保守），梯度越陡 $C_b$ 越大，允許設計彎矩越高。

---

## 二、現行公式（AISC 360 F1-1 / 台灣規範）

**四點法（Kirby-Nethercot 公式）：**

$$\boxed{C_b = \frac{12.5\,M_{max}}{2.5\,M_{max} + 3\,M_A + 4\,M_B + 3\,M_C}}$$

其中（無支撐區間內）：
- $M_{max}$ = 最大彎矩絕對值
- $M_A$ = 1/4 點彎矩絕對值
- $M_B$ = 中點彎矩絕對值
- $M_C$ = 3/4 點彎矩絕對值

**分母展開的物理意義：**

$$2.5\,M_{max} + 3\,M_A + 4\,M_B + 3\,M_C$$

這是對**彎矩圖形狀**進行加權積分的近似：
- 中點 $M_B$ 的權重（4）最高 → 中點彎矩對 LTB 最關鍵
- 兩端點（$M_A$、$M_C$）各 3 → 對稱考慮
- $M_{max}$ 以 2.5 加權 → 確保分母至少 $2.5 M_{max}$（讓 $C_b \leq 12.5/2.5 = 5.0$）

---

## 三、特殊案例的 Cb 值

| 載重型式 | 彎矩圖形狀 | Cb 值 |
|---------|----------|-------|
| 均勻彎矩（兩端等力矩） | 矩形 | **1.0**（最保守） |
| 跨中集中力（簡支） | 三角形 | 1.32 |
| 均佈載重（簡支） | 拋物線 | 1.14 |
| 三分點兩點集中力（簡支） | 梯形 | 1.14 |
| 懸臂梁（端部集中力） | 三角形 | 1.0（規範規定，保守） |
| 力矩比 $M_1/M_2 = 0$（一端彎一端零） | 三角形 | ≈ 1.67 |
| 力矩比 $M_1/M_2 = -1$（反曲率） | S形 | ≈ 2.3（受上限限制） |

> ⚠️ **均勻彎矩 Cb 不是「最有利」，而是「最不利」**：Cb = 1.0 代表無修正（均勻彎矩條件），是設計中最保守的情形。梯度越大（$C_b > 1$）代表強度越高。

---

## 四、ASD 版本

ASD 法中有另一個公式（出現在舊版 AISC ASD 9th Ed.）：

$$C_b = 1.75 + 1.05\left(\frac{M_1}{M_2}\right) + 0.3\left(\frac{M_1}{M_2}\right)^2 \leq 2.3$$

其中 $M_1/M_2$ = 小端/大端力矩比，**同曲率為正，反曲率為負**。

這個公式只適用於**線性彎矩分布**（兩端力矩決定），對有橫向載重的情形需取 $C_b = 1.0$。

**對比 LRFD 四點法**：
- LRFD 四點法適用任何彎矩分布形狀（含橫向載重）
- ASD 公式只適用於兩端力矩線性分布，且有橫向載重時保守取 1.0

---

## 五、Cb 的使用限制

1. **$C_b$ 放大後不得超過 $M_p$**：$M_n = C_b[\cdots] \leq M_p$，強度上限是截面塑性彎矩。
2. **懸臂梁保守取 1.0**：懸臂梁端部約束不完整，即使彎矩有梯度，規範仍要求 $C_b = 1.0$（保守）。
3. **無支撐長度 vs 彎矩形狀**：$C_b$ 修正的是「彎矩形狀的影響」，不修正無支撐長度本身。

---

## 六、常見考試問法

| 考法 | 答題要點 |
|------|---------|
| 「Cb 的物理意義」 | 彎矩梯度相對於均勻彎矩的強度修正倍數；梯度越陡、LTB 越不容易發生 |
| 「為何均勻彎矩 Cb=1.0 最不利」 | 均勻彎矩讓整個無支撐段每截面都承受相同大彎矩，LTB 臨界彎矩最低 |
| 「四點法的係數為何是 2.5/3/4/3」 | 加權積分近似彎矩圖形狀，中點權重最高（最影響 LTB 穩定性） |
