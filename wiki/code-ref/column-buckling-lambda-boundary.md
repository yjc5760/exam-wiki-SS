# 壓力桿件：λ_c = 1.5 彈性與非彈性挫曲分界點

**規範來源：** AISC 360 Chapter E（壓力桿件）  
**關聯知識點：** [[lrfd-column]]、[[compression-member]]、[[EULER-BUCKLING]]

---

## ⭐ 核心記憶點：λ_c = 1.5 是分界點，必記！

$$\lambda_c = \frac{KL}{r\pi}\sqrt{\frac{F_y}{E}}$$

| 範圍 | 挫曲型態 | $F_{cr}$ 公式 |
|------|---------|-------------|
| $\lambda_c \leq 1.5$ | **非彈性挫曲**（inelastic buckling） | $F_{cr} = \left[\exp(-0.419\,\lambda_c^2)\right] F_y$ |
| $\lambda_c > 1.5$ | **彈性挫曲**（elastic / Euler buckling） | $F_{cr} = \dfrac{0.877}{\lambda_c^2}\,F_y$ |

> 等效寫法（兩者相同）：$\exp(-0.419\lambda_c^2) = 0.658^{\lambda_c^2}$，因為 $\ln 0.658 \approx -0.419$。

---

## 分界點數值：0.39Fy

在 $\lambda_c = 1.5$ 時，兩條公式交接，臨界壓應力為：

$$F_{cr}\big|_{\lambda_c=1.5} = \exp(-0.419 \times 1.5^2) \times F_y = \exp(-0.943) \times F_y \approx 0.39\,F_y$$

**驗算（彈性公式）：**
$$F_{cr} = \frac{0.877}{1.5^2} F_y = \frac{0.877}{2.25} F_y = 0.390\,F_y \checkmark$$

兩式在 $\lambda_c = 1.5$ 處連續，如圖所示。

## Fcr–λc 曲線示意

```
Fcr
 |
Fy ──┐
     │   ← 非彈性挫曲區（λc ≤ 1.5）
     │    Fcr = exp(-0.419λc²) Fy
     │
0.39Fy ─ ─ ─ ─ ─ ─ ─ ─ ✕ ← 分界點 (λc = 1.5)
                          │    ← 彈性挫曲區（λc > 1.5）
                          │     Fcr = (0.877/λc²) Fy
                          ──────────────────→ λc
                        1.5
```

## 物理意義

| 區域 | 物理行為 | 控制因素 |
|------|---------|---------|
| $\lambda_c \leq 1.5$（短柱/中柱） | 挫曲前已發生部分降伏，**殘留應力**使斷面提早喪失剛度 | 材料強度 $F_y$ + 殘留應力 |
| $\lambda_c > 1.5$（長柱） | 挫曲前仍為彈性，由 **Euler 理論**主宰 | 彈性模數 $E$（與 $F_y$ 無關） |

> 長柱 Euler 公式中，0.877 是折減係數（≈ 1/1.14²），考慮初始幾何不完美（初始偏心）的影響。

---

## 考試速查：如何判斷用哪條公式

1. 算出 $\lambda_c = \frac{KL}{r\pi}\sqrt{\frac{F_y}{E}}$
2. 比較與 1.5 的大小：
   - $\lambda_c \leq 1.5$ → 用指數公式（非彈性）
   - $\lambda_c > 1.5$ → 用 Euler 折減公式（彈性）
3. 代入 $\phi_c P_n = 0.85 \times F_{cr} \times A_g$

> 考試題目中 $\lambda_c$ 通常落在 0.3–1.2 之間（非彈性區），純彈性長柱較少出現。
