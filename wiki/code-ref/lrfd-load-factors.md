# LRFD 載重係數 1.2D + 1.6L 的來源

**Code-Ref ID：** lrfd-load-factors
**規範來源：** AISC 360 / ASCE 7 / LRFD 可靠度理論

---

## 一、LRFD 核心設計方程

$$\phi R_n \geq \gamma_D D + \gamma_L L = 1.2 D + 1.6 L$$

其中：
- $D$ = 靜載重效應（Dead Load）
- $L$ = 活載重效應（Live Load）
- $\gamma_D = 1.2$，$\gamma_L = 1.6$

---

## 二、為何 γD = 1.2 而 γL = 1.6？

**核心邏輯：不確定性越大 → 載重係數越大**

| 載重類型 | 不確定性來源 | 變異係數 $V_Q$ | 係數 |
|---------|------------|-------------|------|
| 靜載重 $D$ | 材料密度已知、尺寸可精確量測 | 低（約 0.10） | **1.2** |
| 活載重 $L$ | 使用型態多變、難以預測最大值 | 高（約 0.25～0.35） | **1.6** |

**統計基礎（LRFD 可靠度校準）：**

載重效應 $Q$ 設定為在 50 年使用期內，有 5% 機率被超越的值（第 95 百分位）：

$$\gamma = 1 + \alpha_Q \cdot \beta_{target} \cdot V_Q$$

- 靜載重變異係數低 → 係數 1.2
- 活載重變異係數高 → 係數 1.6

## 三、常用載重組合

| 組合 | 公式 | 說明 |
|------|------|------|
| 重力（常用） | $1.2D + 1.6L$ | 多數設計控制工況 |
| 重力（輕活載重） | $1.4D$ | 活載重很小時 |
| 重力 + 雪 | $1.2D + 1.6S + 0.5L$ | — |
| 重力 + 風 | $1.2D + 1.0W + 1.0L$ | 風載重為主時 |
| 地震 | $1.2D + 1.0E + 1.0L$ | — |
| 強度下限 | $0.9D + 1.0W$ | 傾覆驗算（D 減少最不利） |

---

## 四、ASD 與 LRFD 的對應關係

ASD 服務載重：
$$P_a = D + L$$

LRFD 因數化載重（L 遠大於 D 的極端情況）：
$$P_u = 1.2D + 1.6L \xrightarrow{L \gg D} 1.6(D + L) = 1.6 P_a$$

**這就是「ASD 直接分析法中載重乘以 1.6」的來源**（詳見 [[asd-1.6-coefficient]]）：

當活載重主控設計時，LRFD 因數化載重 ≈ ASD 服務載重 × 1.6。

若靜載重主控（$L = 0$）：
$$P_u = 1.2 D = 1.2 P_a$$

因此 1.6 是保守的上限值，規範選用 1.6 以涵蓋所有情況。

## 五、常見考試問法

| 問法 | 要點 |
|------|------|
| 「為何 γL > γD？」 | 活載重不確定性（變異係數）高於靜載重 |
| 「係數 1.6 的物理意義」 | 活載重統計上有 5% 機率超越標稱值 1.6 倍 |
| 「1.2D + 1.6L 的來源」 | LRFD 可靠度理論；以目標可靠度指標 β=3.0 校準 |
| 「ASD 的 1.6 倍怎來」 | Pu/Pa 在 L>>D 時趨近 1.6，AISC 取最保守上限 |
