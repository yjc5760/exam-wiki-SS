## 公式全貌

$$\text{當 } \frac{P_u}{\phi_c P_n} \geq 0.2：\quad \frac{P_u}{\phi_c P_n} + \frac{8}{9}\!\left(\frac{M_{ux}}{\phi_b M_{nx}} + \frac{M_{uy}}{\phi_b M_{ny}}\right) \leq 1.0 \quad \cdots \text{H1-1a}$$

$$\text{當 } \frac{P_u}{\phi_c P_n} < 0.2：\quad \frac{P_u}{2\phi_c P_n} + \left(\frac{M_{ux}}{\phi_b M_{nx}} + \frac{M_{uy}}{\phi_b M_{ny}}\right) \leq 1.0 \quad \cdots \text{H1-1b}$$

---

## 物理意義：「強度利用率之和 ≤ 1」

每一項都是「**實際需求 / 單獨極限強度**」的比值，代表各作用力對斷面強度的「消耗比例」：

| 項目 | 物理意義 |
|------|---------|
| $P_u / \phi_c P_n$ | **軸力利用率**：實際軸壓力佔純壓強度的比例 |
| $M_{ux} / \phi_b M_{nx}$ | **強軸彎矩利用率**：實際強軸彎矩佔純撓曲強度的比例 |
| $M_{uy} / \phi_b M_{ny}$ | **弱軸彎矩利用率**：實際弱軸彎矩佔純撓曲強度的比例 |

> **核心概念：** 斷面的「強度資源」是有限的。軸力吃掉一部分，剩餘才給彎矩用。
> 兩者的總消耗不可超過 1.0，否則桿件失效。

## P-M 互制圖幾何解讀

```
Pu
│
φcPn ─────●  ← 純壓（無彎矩）
          │ ╲  H1-1a（斜率較陡）
          │  ╲
0.2φcPn ─ ─ ─●─────────────────●  ← 銜接點（kink）
              ╲  H1-1b（斜率較平）
               ╲
                ●──────────────→ Mu
             0.9φbMn  φbMn
                       ↑ 純彎（無軸力）
```

三個關鍵點：
1. **頂點 $(0,\;\phi_c P_n)$：** 純壓，無彎矩
2. **銜接點 $(0.9\phi_b M_n,\;0.2\phi_c P_n)$：** H1-1a 與 H1-1b 的連接處
3. **底點 $(\phi_b M_n,\;0)$：** 純彎，無軸力

包絡線以內 → 安全；包絡線以外 → 超過強度 → 失效。

---

## 銜接點的推導：為什麼是 0.9φbMn？

當 $P_u = 0.2\phi_c P_n$（代入 H1-1a）：

$$0.2 + \frac{8}{9} \cdot \frac{M_u}{\phi_b M_n} = 1.0$$

$$\frac{8}{9} \cdot \frac{M_u}{\phi_b M_n} = 0.8 \quad \Rightarrow \quad \frac{M_u}{\phi_b M_n} = 0.8 \times \frac{9}{8} = 0.9$$

當 $P_u = 0.2\phi_c P_n$（代入 H1-1b）：

$$\frac{0.2\phi_c P_n}{2\phi_c P_n} + \frac{M_u}{\phi_b M_n} = 1.0 \quad \Rightarrow \quad 0.1 + \frac{M_u}{\phi_b M_n} = 1.0 \quad \Rightarrow \quad \frac{M_u}{\phi_b M_n} = 0.9$$

✅ 兩式在 $P_u = 0.2\phi_c P_n$ 時得到相同的 $M_u = 0.9\phi_b M_n$，**完全連續，無跳躍**。

## 為什麼要分兩段？各段的物理行為

### H1-1a（上段，$P_u/\phi_c P_n \geq 0.2$）：**軸力主控區**

- 桿件以「柱」的行為為主，軸壓佔主導
- 軸力項係數為 1（全額計入），彎矩項係數為 8/9（略有折減）
- **8/9 的幾何作用：** 使直線從 $(\phi_c P_n, 0)$ 連接到銜接點 $(0.9\phi_b M_n, 0.2\phi_c P_n)$，維持連續性

### H1-1b（下段，$P_u/\phi_c P_n < 0.2$）：**彎矩主控區**

- 桿件以「梁」的行為為主，軸力影響較小
- 軸力項分母變為 $2\phi_c P_n$（係數折半），表示輕微軸力對互制的貢獻減半
- 彎矩項係數回到 1（不折減），可發揮完整彎矩強度至 $\phi_b M_n$

> **直覺：** 軸力很小（< 20%）時，就算完全忽略軸力，誤差也不大。H1-1b 就是「以梁為主」的簡化形式。

## 為什麼是「雙折線」而非曲線？

梁柱的真實 P-M 包絡線是**向外凸的曲線**（如橢圓型），AISC 用雙折線取代，原因：

1. **保守性：** 雙折線在曲線的內側，永遠偏安全
2. **計算便利：** 線性方程直接代入求解，不需解非線性方程
3. **足夠準確：** 對 H 型斷面，雙折線誤差在 5% 以內，工程上可接受

## 考試常見應用邏輯

```
Step 1：算出 Pu, Mux, Muy（設計載重，已含 B1 放大）
Step 2：算出 φcPn, φbMnx, φbMny（桿件強度）
Step 3：判斷門檻
    Pu / φcPn ≥ 0.2 → 用 H1-1a（含 8/9）
    Pu / φcPn < 0.2 → 用 H1-1b（軸力項 ÷2）
Step 4：代入對應方程式，檢核 ≤ 1.0，或解未知量
```

## 數字小結（背起來）

| 數字 | 意義 |
|:----:|------|
| **0.2** | 軸力利用率的門檻，決定用哪條方程式 |
| **8/9 ≈ 0.889** | H1-1a 中彎矩項的折減係數（為維持銜接點連續性） |
| **0.9** | 銜接點的彎矩利用率（$P_u = 0.2\phi_c P_n$ 時的最大彎矩比） |
| **1/2** | H1-1b 中軸力項的折減（分母乘以 2） |
