# 剪力遲滯 U 值公式的來源

**Code-Ref ID：** shear-lag-u-basis
**對應方法頁：** [[shear-lag-u]]（`wiki/methods/shear-lag-u.md`）
**對應概念頁：** [[SHEAR-LAG]]

---

## 一、剪力遲滯的物理現象

拉力構材只有部分肢（板）連接時，**接觸面上的剪力無法瞬間均勻傳遞到整個截面**，造成未直接連接的板件應力落後（遲滯）於直接連接板件。

**結果：** 破壞時，有效承力面積 < 全淨面積 $A_n$。

例：
- 角鋼只以一肢連接螺栓 → 另一肢的應力在連接端附近明顯低於直接連接肢
- 全截面均勻受力（直接連接兩肢）→ 無剪力遲滯，$U = 1.0$

---

## 二、Munse & Chesson（1963）的試驗基礎

美國學者 Munse 和 Chesson 在大量拉力試件試驗後，發現「剪力遲滯的嚴重程度」主要與以下兩個幾何參數有關：

1. **$\bar{x}$（截面形心到連接面的距離）**：$\bar{x}$ 越大，未連接部分離連接面越遠，遲滯越嚴重
2. **$L$（連接長度）**：連接越長，力量傳遞越充分，遲滯越小

---

## 三、U 值公式推導

AISC 規範將大量試驗資料迴歸，得出有效截面效率 $U$ 的簡化公式：

$$\boxed{U = 1 - \frac{\bar{x}}{L}}$$

其中：
- $\bar{x}$（x-bar）= 截面形心到**連接面（gusset 板或接合面）** 的垂直距離
- $L$ = 連接長度（端距最外排螺栓到最末排螺栓的距離；或銲縫長度）

**直觀驗證：**

| 情境 | $\bar{x}/L$ | $U$ |
|------|------------|-----|
| $\bar{x} = 0$（形心就在連接面上） | 0 | 1.0（無遲滯） |
| $\bar{x}/L = 0.1$ | 0.1 | 0.9 |
| $\bar{x}/L = 0.5$ | 0.5 | 0.5（嚴重遲滯） |
| $L$ 極大（$\bar{x}/L \to 0$） | ≈ 0 | ≈ 1.0（長連接 → 幾乎無遲滯） |

---

## 四、實務中的 U 值表格（AISC Table D3.1）

規範同時提供**查表法**（當無法精確計算 $\bar{x}$ 時）：

| 構材類型 | 連接條件 | U |
|---------|---------|---|
| 全截面連接 | — | 1.0 |
| W 型鋼 | 只連接翼板（$b_f \geq 2/3 d$） | 0.90 |
| W 型鋼 | 只連接翼板（$b_f < 2/3 d$） | 0.85 |
| W 型鋼 | 只連接腹板 | 0.70 |
| 角鋼 | 4排+ 螺栓 | 0.80 |
| 角鋼 | 3排螺栓 | 0.60 |
| 槽鋼（C 型） | 3排+ 螺栓 | 0.90 |

**考試策略：** 若題目給出斷面尺寸可算 $\bar{x}$，用公式法；若只給出構材類型，查表。**公式法較精確，查表法為保守下限**。

---

## 五、有效淨面積

$$A_e = U \cdot A_n$$

拉力破壞強度：
$$\phi_t P_n = \phi_t F_u A_e = 0.75 \times F_u \times U \times A_n$$

---

## 六、常見考試問法

| 問法 | 要點 |
|------|------|
| 「U 值公式的推導依據」 | Munse & Chesson 試驗迴歸；$\bar{x}$ 越大/連接越短 → 遲滯越嚴重 → U 越小 |
| 「$\bar{x}$ 怎麼量？」 | 截面形心到連接面（螺栓所在的肢面）的垂直距離 |
| 「連接長度 $L$ 定義」 | 最外排螺栓到最末排螺栓中心距；銲縫長度 |
| 「等肢角鋼兩肢均連接的 U」 | $\bar{x} = 0$（形心在連接平面） → $U = 1.0$ |
