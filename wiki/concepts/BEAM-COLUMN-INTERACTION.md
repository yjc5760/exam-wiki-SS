# 梁柱互制行為

**概念 ID：** BEAM-COLUMN-INTERACTION
**知識分類：** Part 1.3 梁柱桿件
**規範來源：** 鋼結構設計規範 第八章

## 定義

構件在同時承受軸力與彎矩時，兩者會相互影響，降低構件承載能力的現象。軸壓力會放大彎矩產生的側向變形（P-δ 效應），而彎矩會使斷面應力提早達到降伏，降低軸向承載力。設計時必須使用互制公式進行檢核。

## 關鍵公式

**LRFD 梁柱互制公式（H1-1）：**

$$\frac{P_u}{\phi_c P_n} \geq 0.2 \text{ 時：}$$

$$\frac{P_u}{\phi_c P_n} + \frac{8}{9}\left(\frac{M_{ux}}{\phi_b M_{nx}} + \frac{M_{uy}}{\phi_b M_{ny}}\right) \leq 1.0$$

$$\frac{P_u}{\phi_c P_n} < 0.2 \text{ 時：}$$

$$\frac{P_u}{2\phi_c P_n} + \left(\frac{M_{ux}}{\phi_b M_{nx}} + \frac{M_{uy}}{\phi_b M_{ny}}\right) \leq 1.0$$

**ASD 梁柱互制公式：**

$$\frac{f_a}{F_a} + \frac{C_{mx} f_{bx}}{(1 - f_a/F'_{ex})F_{bx}} + \frac{C_{my} f_{by}}{(1 - f_a/F'_{ey})F_{by}} \leq 1.0$$

（使用性極限狀態公式另有規定）

## 常見陷阱

1. **$P_u / \phi_c P_n$ 的 0.2 判斷**：兩條公式對應軸力比大/小於 0.2，必須先判斷再套用。
2. **二階彎矩 $M_u$**：公式中的彎矩需為放大後的二階彎矩（$M_u = B_1 M_{nt} + B_2 M_{lt}$），而非一階分析直接結果。
3. **ASD 的 $C_m$ 係數**：ASD 用 $C_m$ 修正彎矩項，對應 LRFD 的 $B_1$ 放大係數（概念相同）。
