# 彎矩梯度修正係數 (Cb)

**概念 ID：** BENDING-MODIFICATION-FACTOR-CB
**知識分類：** Part 1.2 撓曲桿件
**規範來源：** 鋼結構極限設計法 §7.8

## 定義

用於修正梁在非均勻彎矩下的側向扭轉挫屈強度。非均勻的彎矩分佈對梁的挫屈有束制效果，$C_b$ 值大於 1.0 代表此提升效應。均勻彎矩（等彎矩段）時 $C_b = 1.0$，此為最保守情形。

## 關鍵公式

**$C_b$ 計算公式（四分點法）：**

$$C_b = \frac{12.5 M_{max}}{2.5 M_{max} + 3 M_A + 4 M_B + 3 M_C}$$

其中：
- $M_{max}$：無支撐段內的最大彎矩絕對值
- $M_A$、$M_B$、$M_C$：無支撐段四分之一、二分之一、四分之三處的彎矩絕對值

**常見載重型式的 $C_b$ 值：**

| 載重型式 | $C_b$ 近似值 |
|---------|------------|
| 均勻彎矩（等彎矩） | 1.00 |
| 單端彎矩（另一端為零） | 1.67 |
| 中點集中載重（簡支） | 1.32 |
| 均佈載重（簡支） | 1.14 |
| 三分點集中對稱載重 | 1.01 |

**設計應用：**

$$\phi_b M_n = C_b \cdot (\phi_b M_n)_{C_b=1} \leq \phi_b M_p$$

## 常見陷阱

1. **三分點對稱載重 $C_b \approx 1.0$**：三分點等彎矩段（中段）$C_b = 1.0$，因為中間段彎矩幾乎均勻。
2. **$C_b$ 不能使 $M_n > M_p$**：$C_b$ 修正後的強度不得超過塑性彎矩 $M_p$（上限）。
3. **使用圖表時確認 $C_b = 1.0$ 條件**：設計輔助圖通常繪於 $C_b = 1.0$，若實際 $C_b > 1.0$ 可向上修正。
