# 塊狀剪力撕裂

**概念 ID：** BLOCK-SHEAR-RUPTURE
**知識分類：** Part 1.4 接合設計
**規範來源：** 鋼結構設計規範 第十章

## 定義

在接頭區域，一種由剪力面破壞與張力面破壞組合而成的複合破壞模式。通常發生在螺栓或銲接接頭的端部，一塊材料像被「撕開」一樣脫離母材。

## 關鍵公式

**LRFD 塊狀剪力強度（$\phi = 0.75$）：**

$$\phi R_n = \phi \left(0.6 F_u A_{nv} + U_{bs} F_u A_{nt}\right) \leq \phi \left(0.6 F_y A_{gv} + U_{bs} F_u A_{nt}\right)$$

其中：
- $A_{gv}$：剪力破壞面的毛斷面積
- $A_{nv}$：剪力破壞面的淨斷面積
- $A_{nt}$：張力破壞面的淨斷面積
- $U_{bs}$：塊狀剪力係數（均勻張力時 = 1.0；非均勻時 = 0.5）

**破壞模式判斷：**
- 剪力破壞（沿螺栓排列方向）+ 張力破壞（垂直螺栓排列方向） = 塊狀剪力
- 取公式中上界與主體的最小值作為設計強度

## 常見陷阱

1. **$U_{bs}$ 選用**：接頭承受均勻張力（如板片接合）$U_{bs} = 1.0$；梁端腹板角板（剪力流不均勻）$U_{bs} = 0.5$。
2. **同時檢核兩種破壞模式**：可能有多種「塊狀」組合，需找出控制性破壞模式（最小強度）。
3. **螺孔孔徑加算**：計算淨斷面積時需扣除螺孔直徑（實際孔徑 = 螺栓直徑 + 3.2 mm 或 1/16"）。
