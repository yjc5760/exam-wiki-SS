# 合成柱

**概念 ID：** COMPOSITE-COLUMN
**知識分類：** Part 2.2 材料科學（合成結構）
**規範來源：** 鋼結構極限設計法 第九章

## 定義

指由型鋼（或鋼管）與混凝土結合而成的壓力構件，常見型式有混凝土包覆型鋼（SRC）與鋼管混凝土（CFT）。此類構件能結合鋼材的韌性與混凝土的抗壓及勁度，有效提高承載能力與防火性能。

## 關鍵公式

**混凝土包覆型鋼柱（SRC）設計強度（LRFD）：**

$$\phi_c P_n = \phi_c A_s F_{my} + \phi_c A_{sr} F_y + \phi_c A_c f'_c$$

等效材料強度（簡化法）：

$$F_{my} = F_y + 0.85 f'_c \frac{A_c}{A_s} + F_{yr}\frac{A_{sr}}{A_s}$$

等效彈性模數：

$$E_m = E_s + 0.2 E_c \frac{A_c}{A_s}$$

**鋼管混凝土柱（CFT）強度：**

$$\phi_c P_n = \phi_c \left(A_s F_y + C_2 f'_c \left(A_c + C_3 \frac{A_{sr} E_s}{E_c}\right)\right)$$

其中 $C_2 = 0.85$（圓管）或 $0.80$（方管）。

## 常見陷阱

1. **SRC 柱的最小型鋼面積比**：型鋼斷面積須佔總斷面積的 4% 以上，才能作為合成柱設計（否則按 RC 設計）。
2. **等效細長比的計算**：合成柱需使用等效材料特性（$F_{my}$、$E_m$）計算 $\lambda_c$，再套用柱強度曲線。
3. **CFT 有圍束效應**：圓形 CFT 的混凝土受三向圍束，強度提升係數不同於矩形管。
