# 鋼結構設計術語速查表（GLOSSARY）

> 詳細概念說明見各 `wiki/concepts/[CONCEPT-ID].md`。

## 符號索引

| 符號 | 名稱 | 定義 / 說明 |
|------|------|------------|
| $F_y$ | 降伏應力 | 鋼材最小規定降伏強度，LRFD/ASD 設計基準 |
| $F_u$ | 極限拉伸應力 | 鋼材最小規定極限強度，用於淨斷面破壞 |
| $E$ | 彈性模數 | 鋼材 $E = 2,040 \text{ tf/cm}^2$（SI: 200 GPa） |
| $G$ | 剪切模數 | $G = 790 \text{ tf/cm}^2$（SI: 77 GPa） |
| $\phi$ | 強度折減係數 | LRFD 設計法：$\phi_b = 0.9$（彎曲）、$\phi_c = 0.85$（壓力）、$\phi_v = 0.9$（剪力） |
| $\Omega$ | 安全係數 | ASD 設計法：$\Omega_b = 1.67$（彎曲）、$\Omega_c = 1.67$（壓力） |

## 斷面性質

| 符號 | 名稱 | 說明 |
|------|------|------|
| $A_g$ | 毛斷面積 | 未扣孔洞的全截面積 |
| $A_n$ | 淨斷面積 | 扣除孔洞後的面積 |
| $A_e$ | 有效淨面積 | $A_e = U \cdot A_n$，考慮剪力遲滯 |
| $I_x, I_y$ | 慣性矩（強軸/弱軸） | 對主軸的截面二次矩 |
| $S_x$ | 彈性斷面模數 | $S_x = I_x / c$，彈性範圍用 |
| $Z_x$ | 塑性斷面模數 | 塑性中性軸兩側面積矩之和，$M_p = F_y Z_x$ |
| $r_x, r_y$ | 迴轉半徑 | $r = \sqrt{I/A}$ |
| $J$ | 聖維南扭轉常數 | 開放斷面的純扭轉剛度 |
| $C_w$ | 翹曲常數 | 雙對稱 I 型：$C_w = I_y h_o^2 / 4$ |
| $r_{ts}$ | LTB 等效半徑 | $r_{ts}^2 = \sqrt{I_y C_w} / S_x$ |
| $f$ | 形狀因數 | $f = Z_x / S_x$，熱軋 H 型鋼約 1.12–1.15 |

## 壓力桿件

| 符號 | 名稱 | 說明 |
|------|------|------|
| $K$ | 有效長度係數 | 依端部束制條件，兩端固接理論值 $K=0.5$ |
| $KL/r$ | 細長比 | 取強軸、弱軸最大值 |
| $\lambda_c$ | 無因次細長比（LRFD） | $\lambda_c = \frac{KL}{r\pi}\sqrt{\frac{F_y}{E}}$ |
| $F_{cr}$ | 臨界壓應力 | 柱強度公式計算值 |
| $F_e$ | 歐拉應力 | $F_e = \pi^2 E / (KL/r)^2$ |
| $\phi_c P_n$ | 設計壓力強度（LRFD） | $\phi_c = 0.85$，$P_n = F_{cr} A_g$ |
| $P_a$ | 容許壓力（ASD） | $P_a = P_n / \Omega_c$ |
| $C_c$ | 彈性/非彈性界限細長比（ASD） | $C_c = \sqrt{2\pi^2 E / F_y}$ |

## 梁桿件（撓曲構材）

| 符號 | 名稱 | 說明 |
|------|------|------|
| $M_p$ | 塑性彎矩 | $M_p = F_y Z_x$，完全塑性截面 |
| $M_y$ | 初始降伏彎矩 | $M_y = F_y S_x$ |
| $L_b$ | 側向無束制長度 | 相鄰側向支撐點間距 |
| $L_p$ | 塑性限制長度 | $L_b \leq L_p$ 時為塑性區（$\phi_b M_n = \phi_b M_p$） |
| $L_r$ | 非彈性/彈性界限長度 | $L_b > L_r$ 時為彈性 LTB |
| $C_b$ | 彎矩梯度修正係數 | 非均勻彎矩時提高名義強度，均勻彎矩 $C_b = 1.0$ |
| $M_{cr}$ | 彈性 LTB 臨界彎矩 | $M_{cr} = C_b \frac{\pi}{L_b}\sqrt{EI_y GJ + \left(\frac{\pi E}{L_b}\right)^2 I_y C_w}$ |
| $\phi_b M_n$ | 設計彎矩強度（LRFD） | 依 $L_b$ 三段判斷（塑性/非彈性/彈性 LTB） |
| $V_n$ | 名義剪力強度 | 依 $h/t_w$ 三段：降伏剪力 → 非彈性 → 張力場 |

**LTB 三段判斷：**

$$\phi_b M_n = \begin{cases}
\phi_b M_p & L_b \leq L_p \text{（塑性區）}\\
\phi_b \left[M_p - (M_p - 0.7F_y S_x)\frac{L_b - L_p}{L_r - L_p}\right] & L_p < L_b \leq L_r \text{（非彈性LTB）}\\
\phi_b F_{cr} S_x \leq \phi_b M_p & L_b > L_r \text{（彈性LTB）}
\end{cases}$$

## 梁柱桿件

| 符號 | 名稱 | 說明 |
|------|------|------|
| $P_u / \phi_c P_n$ | 軸力比 | 判斷使用哪一組 P-M 互制公式的門檻（0.2） |
| $B_1$ | 一階效應放大係數 | 無側移構架，考慮構材曲率放大 $\delta$ 效應 |
| $B_2$ | 側移效應放大係數 | 側移構架，考慮整體 $\Delta$ 效應 |
| $C_m$ | 等效彎矩係數 | $B_1$ 計算用，均勻彎矩 $C_m = 1.0$ |
| $P_e$ | 歐拉挫屈載重 | $P_e = \pi^2 EI / (KL)^2$ |

**P-M 互制公式（LRFD）：**

$$\frac{P_u}{\phi_c P_n} \geq 0.2：\quad \frac{P_u}{\phi_c P_n} + \frac{8}{9}\left(\frac{M_{ux}}{\phi_b M_{nx}} + \frac{M_{uy}}{\phi_b M_{ny}}\right) \leq 1.0$$

$$\frac{P_u}{\phi_c P_n} < 0.2：\quad \frac{P_u}{2\phi_c P_n} + \left(\frac{M_{ux}}{\phi_b M_{nx}} + \frac{M_{uy}}{\phi_b M_{ny}}\right) \leq 1.0$$

## 接合設計

| 符號 | 名稱 | 說明 |
|------|------|------|
| $U$ | 剪力遲滯係數 | 拉力桿件，$A_e = U A_n$，依接合條件查表 |
| $A_{gv}, A_{nv}$ | 剪力毛/淨面積 | 塊狀剪力撕裂用 |
| $A_{gt}, A_{nt}$ | 拉力毛/淨面積 | 塊狀剪力撕裂用 |
| $D$ | 填角銲有效喉深係數 | $D$ = 銲腳尺寸（1/16 in 為單位）× 0.707 |
| $F_{nw}$ | 銲材名義強度 | E70XX 銲條：$F_{nw} = 0.6 \times 70 = 42 \text{ ksi}$ |

## 設計法對照

| 項目 | LRFD（極限設計法） | ASD（容許應力設計法） |
|------|-------------------|---------------------|
| 設計條件 | $\phi R_n \geq \sum \gamma_i Q_i$ | $R_n / \Omega \geq \sum Q_i$ |
| 載重組合 | $1.2D + 1.6L$（控制） | $D + L$（控制） |
| 彎曲折減係數 | $\phi_b = 0.9$ | $\Omega_b = 1.67$ |
| 壓力折減係數 | $\phi_c = 0.85$ | $\Omega_c = 1.67$ |
| 剪力折減係數 | $\phi_v = 0.9$ | $\Omega_v = 1.67$ |

## 耐震設計關鍵術語

| 術語 | 說明 |
|------|------|
| 強柱弱梁（SCWB） | $\sum M_{pc} \geq 1.0 \sum M_{pb}$，塑鉸形成於梁端 |
| Panel Zone（節點域） | 梁柱接頭的柱腹板區域，控制剪力強度 |
| RBS（犬骨接頭） | Reduced Beam Section，削減梁翼板以移近塑鉸位置 |
| 容量設計 | 以預定破壞模式（梁端塑鉸）設計，確保其他構件不先破壞 |
| SN 鋼材 | 日本耐震鋼材（SN400B/SN490B），限制降伏比 $F_y/F_u \leq 0.8$ |
| 降伏比 | $F_y / F_u$，越低表示延性越好 |
| 超強係數 $R_y$ | 實際降伏強度與最小規定值之比，用於容量設計放大 |
