# 側向扭轉挫屈 (LTB)

**概念 ID：** LATERAL-TORSIONAL-BUCKLING
**知識分類：** Part 1.2 撓曲桿件
**規範來源：** 鋼結構極限設計法 第七章 撓曲構材

## 定義

受強軸彎曲的梁桿件，其受壓翼板如同軸壓柱，當無支撐長度過長時，可能發生側向彎曲並伴隨斷面扭轉的「挫屈」現象，導致其承載能力下降。這種不穩定行為稱為側向扭轉挫屈。

## 關鍵公式

**三區段設計強度（LRFD，$\phi_b = 0.9$）：**

$$L_p = \frac{80 r_y}{\sqrt{F_y}} \quad \text{（塑性區上限，tf/cm²）}$$

$$L_r = \frac{r_y X_1}{\sqrt{F_y - F_r}} \sqrt{1 + \sqrt{1 + X_2(F_y - F_r)^2}}$$

| 無支撐長度 | 設計強度 $\phi_b M_n$ |
|-----------|----------------------|
| $L_b \leq L_p$ | $\phi_b M_p = \phi_b Z_x F_y$（塑性區） |
| $L_p < L_b \leq L_r$ | 線性內插（非彈性 LTB 區） |
| $L_b > L_r$ | 彈性 LTB（歐拉型挫屈） |

**彎矩梯度修正：** $\phi_b M_n \leq C_b \cdot (\phi_b M_n)_{C_b=1} \leq \phi_b M_p$

## 常見陷阱

1. **$L_b$ 誤用梁全長**：側向支撐點（支承點、載重點）之間距離才是 $L_b$，非梁總跨距。
2. **圖讀後未計算驗算**：$C_b = 1.0$ 的圖表讀值與精確 $\phi_b M_p$ 可能相差 1–2%，必須計算確認。
3. **忘記乘以 $C_b$**：非均勻彎矩時 $C_b > 1.0$，可提升設計強度。
4. **$Z_x$ 計算錯誤**：翼板面積 × 形心間距 + 腹板貢獻，分子分母勿混淆。
