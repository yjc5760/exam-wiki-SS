# 使用性撓度限制

**概念 ID：** SERVICEABILITY-DEFLECTION
**知識分類：** Part 1.2 撓曲桿件 / 使用性設計
**規範來源：** 鋼結構設計規範 §4.7 / AISC 360 Commentary L

## 定義

**使用性設計（Serviceability Design）** 是極限狀態設計的第二類檢核：在結構強度滿足後，確保在服務載重下撓度不超過規範限值，避免影響使用功能（如天花板龜裂、外觀不雅、機械設備干涉）。

## 撓度限值

| 條件 | 限值 | 說明 |
|------|------|------|
| 梁承受活載重 | $\delta_L \leq L/360$ | 防止脆性隔間牆損壞（最常考） |
| 梁承受全部載重 | $\delta_{total} \leq L/240$ | 一般使用性要求 |
| 屋頂梁（無天花板） | $\delta \leq L/180$ | 較寬鬆 |
| 柱側向漂移（層間） | $\Delta \leq h/400 \sim h/500$ | 視規範版本而定 |

> ⚠️ **關鍵：撓度計算用服務載重（未因數化），不是設計載重**

## 常用撓度公式

| 載重型式 | 公式 |
|---------|------|
| 均佈載重 $w$，簡支梁 | $\delta = \dfrac{5wL^4}{384EI}$ |
| 跨中集中力 $P$，簡支梁 | $\delta = \dfrac{PL^3}{48EI}$ |
| 三分點對稱兩集中力 $P$ | $\delta = \dfrac{23PL^3}{648EI}$ |
| 懸臂梁端部集中力 $P$ | $\delta = \dfrac{PL^3}{3EI}$ |

> ⚠️ **常考陷阱（三分點公式）**：跨中兩個集中力（三分點）的撓度公式是 $23PL^3/648EI$（$P$ 為單個力），不可用 $PL^3/48EI$。

## 使用性 vs. 強度設計

| 項目 | 強度設計（LRFD） | 使用性設計 |
|------|---------------|-----------|
| 載重 | 因數化（$1.2D + 1.6L$） | **服務載重（$D + L$）或僅 $L$** |
| 目的 | 避免破壞 | 避免影響使用功能 |
| 允許範圍 | $\phi R_n \geq R_u$ | $\delta \leq \delta_{allow}$ |

## 常見陷阱

1. **誤用因數化載重計算撓度**：使用性撓度必須用未因數化的服務載重（活載重 $P_L$），若用 $P_u = 1.2P_D + 1.6P_L$ 計算會大幅高估撓度。
2. **撓度限值單位一致**：$L$ 和撓度 $\delta$ 必須用相同單位（cm 或 m）。
3. **三分點公式記憶**：$23PL^3/648EI$ 是兩個三分點集中力（各 $P$），不要與其他常見公式混淆。
4. **I 值選取**：撓度計算用**彈性 $I_x$**（慣性矩），不是 $Z_x$ 或 $S_x$。
