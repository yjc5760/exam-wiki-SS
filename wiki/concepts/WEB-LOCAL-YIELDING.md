# 腹板局部降伏

**概念 ID：** WEB-LOCAL-YIELDING
**知識分類：** Part 1.2 撓曲桿件
**規範來源：** 鋼結構設計規範

## 定義

指在梁端支承處或集中載重作用處，因壓力沿腹板以一定斜率擴散，在腹板與翼板交界的角隅處（k 值範圍內）應力過度集中而導致的局部降伏現象。

## 前置概念

（無必要前置概念）

## 相關概念

- WEB-CRIPPLING — 腹板翹曲破壞
- BEARING-STIFFENER — 加勁板（防止腹板局部破壞）

## 關鍵公式

**腹板局部降伏強度（在支承端，$N/d \leq 0.2$）：**

$$\phi R_n = \phi F_{yw} t_w (2.5k + N)$$

**（支承端，$N/d > 0.2$）：**

$$\phi R_n = \phi F_{yw} t_w \left(4N + k\right)$$

**（在集中載重點，非端部）：**

$$\phi R_n = \phi F_{yw} t_w (5k + N)$$

其中：
- $k$：翼板外緣至腹板圓角末端的距離（由斷面表查取）
- $N$：承壓長度（支承板長度）
- $t_w$：腹板厚度
- $\phi = 1.0$（LRFD 腹板局部降伏）

## 常見陷阱

1. **支承端與跨中載重點公式不同**：端部（$2.5k + N$）與跨中（$5k + N$）的擴散角不同。
2. **$k$ 值查斷面表**：$k$ 是製造商型鋼表格中的尺寸參數，不是有效長度係數。
3. **腹板局部降伏 $\phi = 1.0$**：此極限狀態的 $\phi$ 因子為 1.0，不同於一般彎曲/剪力的 0.9。
