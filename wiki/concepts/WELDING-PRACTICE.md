# 銲接施工實務

**概念 ID：** WELDING-PRACTICE
**知識分類：** Part 2.3 施工與品管
**規範來源：** 建築鋼結構施工規範 / AWS D1.1

## 定義

**銲條乾燥**：低氫銲條須在 150°C 以上烘乾，防止氫致冷裂縫（延遲裂縫）。
**氫致冷裂縫**：成因 = 氫源（濕氣）+ 高應力 + 淬硬組織；預防 = 預熱 + 低氫銲條。
**不可塗裝部位**：摩阻型接合面、混凝土包覆部分、現場銲接區（銲前 50mm 內）。
