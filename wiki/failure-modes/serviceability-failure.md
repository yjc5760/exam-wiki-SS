# 使用性失敗模式

> **核心問題：** 結構安全無虞，但功能或舒適性受損。使用性是否「失敗」取決於允許限制，而非材料強度。

---

## 使用性失敗的本質

使用性失敗（Serviceability Failure）≠ 強度或穩定失敗：

| 項目 | 強度/穩定失敗 | 使用性失敗 |
|------|------------|---------|
| 觸發條件 | 超過材料或幾何極限 | 超過功能或舒適限制 |
| 可逆性 | 不可逆 | 通常可逆（移除載重後恢復） |
| 安全性 | 直接威脅 | 不直接威脅生命 |
| 設計載重 | 極限狀態（1.2D+1.6L） | 服務載重（D+L，無放大） |
| 規範類型 | 強制規定 | 建議值（工程判斷） |

---

## 三種使用性失敗型式

| 型式 | 觸發條件 | 常見限制值 |
|------|---------|----------|
| **撓度過大** | 撓度 > 允許撓度 | L/360（活載）、L/240（全載） |
| **振動過強** | 自然頻率或加速度超限 | fn > 4 Hz（一般樓板）；fn > 8 Hz（跳舞場所） |
| **側移過大** | 層間側移 > 允許值 | H/400（風）、H/200（地震） |

---

## 1. 撓度失敗（Excessive Deflection）

### 常用撓度公式（簡支梁）

| 載重型式 | 最大撓度位置 | 公式 |
|---------|-----------|------|
| 均布載重 w | 跨中 | $\delta = \frac{5wL^4}{384EI}$ |
| 集中載重 P（跨中） | 跨中 | $\delta = \frac{PL^3}{48EI}$ |
| 集中載重 P（a, b 點） | 近似跨中 | $\delta_{max} \approx \frac{Pb(3L^2-4b^2)}{48EI}$ |

### 允許撓度（參考 AISC / 台灣規範）

| 用途 | 活載撓度 | 全載撓度 |
|------|---------|---------|
| 一般樓板 | L/360 | L/240 |
| 屋頂（有排水考量） | L/360 | L/240 |
| 支承易裂材料（磁磚）| L/480 | L/360 |
| 橋樑 | — | L/800（常見） |

### 撓度控制設計策略

1. **加大 I（增加斷面深度）**：最有效
2. **預拱（Camber）**：預先反拱 75~80% 的恆載撓度
3. **改為連續梁**：減少有效跨度
4. **組合梁**：加混凝土板大幅提升 I

### 相關考點

- [[SERVICEABILITY-DEFLECTION]]
- 歷屆題目：SS-2010-3, SS-2015-1, SS-2016-3, SS-2022-3

## 2. 振動失敗（Floor Vibration）

### 發生機制

樓板自然頻率 fn 接近人行走激振頻率（1.6~2.2 Hz 的倍頻）→ 共振 → 過大振動加速度。

### 自然頻率估算

$$f_n = \frac{\pi}{2} \sqrt{\frac{g}{\delta_{st}}}$$

- $\delta_{st}$：靜撓度（服務載重下），單位與 g 一致
- $g = 9.81 \text{ m/s}^2$

### 加速度限制（AISC Design Guide 11）

| 使用用途 | 允許加速度 a/g |
|---------|-------------|
| 辦公室 | 0.5% |
| 商業空間 | 1.5% |
| 健身房、跳舞 | 5~7% |
| 機械設備室 | 按廠商規定 |

### 考試出現方式

- 「計算樓板自然頻率，判斷是否滿足振動需求」
- 「說明組合梁如何改善振動問題」（提升 fn）

---

## 3. 側移失敗（Excessive Drift）

### 層間側移角（Drift Ratio / Inter-story Drift）

$$\Delta/H \leq \text{規定限值}$$

- $\Delta$：該層側移量
- $H$：該層層高

### 允許層間側移角

| 載重類型 | 結構型式 | 限值 |
|---------|---------|------|
| 風 | 一般建築 | H/400 ~ H/500 |
| 風 | 對外觀敏感（帷幕牆） | H/500 ~ H/600 |
| 地震（SLS） | SMRF | 0.01H ~ 0.02H |
| 地震（ULS） | SMRF | 0.025H（台灣建築技術規則） |

### 側移控制設計策略

1. **加大柱斷面**（提升側向剛度）
2. **使用剪力牆或斜撐**
3. **調整結構平面配置**（避免扭轉）
4. **二階效應（P-Δ）放大**：側移大時放大係數 B2 顯著增加設計力

### 相關考點

- [[SECOND-ORDER-ANALYSIS]]（P-Δ 效應與側移關係）
- [[SEISMIC-DESIGN-PHILOSOPHY]]（耐震側移限值）
- 歷屆題目：SS-2013-4, SS-2014-4, SS-2023-1

## 使用性設計的特殊考量

### 預拱（Camber）設計

$$\text{Camber} = 0.75 \sim 0.80 \times \delta_{D}$$

- 只針對恆載撓度（D），不含活載
- 目的：使服務載重下樓板維持水平
- **注意**：組合梁的 camber 在混凝土硬化前就存在，需確認施工期間穩定性

### 使用性與強度的載重差異

```
強度設計：LRFD 載重組合
  U = 1.2D + 1.6L（放大係數）

使用性設計：服務載重
  無放大係數：D + L 或 D + 0.5L（依規範）
```
