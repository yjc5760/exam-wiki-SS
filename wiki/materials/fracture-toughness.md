# 斷裂韌性（Fracture Toughness）

> **核心問題：** 鋼材在含缺口或裂縫情況下，如何抵抗脆性斷裂？SN 鋼材如何分級？

---

## 斷裂韌性的本質

一般強度試驗（Fy, Fu）使用**光滑試件**，無法反映實際鋼材含缺口時的行為。

**斷裂韌性**衡量：有缺口存在時，鋼材在斷裂前能吸收的能量。

```
缺口韌性（Notch Toughness）：
    低韌性 → 脆性斷裂（無預警，低溫更危險）
    高韌性 → 有塑性變形才斷裂（延性破壞）
```

---

## CVN 衝擊試驗（Charpy V-Notch）

### 試驗方法

```
單擺衝擊試件（含標準 V 型缺口）
    ↓
擺錘從固定高度落下衝擊試件
    ↓
量測試件斷裂所吸收的能量（單位：Joule 或 ft-lbf）
```

### CVN 與溫度的關係

```
CVN (J)
  │
  │                  ___________（韌性平台，高溫）
  │                 /
  │                / 轉換區
  │_______________/               （脆性平台，低溫）
  │
  └─────────────────────────────── 溫度 (°C)
      -40  -20   0   20   40
```

- **DBTT（延-脆轉換溫度）**：CVN 急降的溫度範圍
- 設計要求：在**最低使用溫度**下，CVN ≥ 規定值

### 台灣規範要求（參考）

| 鋼材等級 | CVN 要求 | 試驗溫度 |
|---------|---------|---------|
| SN400A | 無要求 | — |
| SN400B / SN490B | ≥ 27 J | 0°C |
| SN400C / SN490C | ≥ 27 J | 0°C（+Z向韌性） |

## SN 鋼材分級系統

SN（Steel New / 新型構造用鋼）是台灣與日本針對**耐震結構**開發的鋼材。

### 三級分類（A/B/C）

| 等級 | Fy 上限 | Fy/Fu | CVN | Z向韌性 | 適用場合 |
|------|--------|-------|-----|--------|---------|
| **A 級** | 無控制 | 無要求 | 無要求 | 無 | 一般非結構部位 |
| **B 級** | 有上限 | ≤ 0.80 | 27J@0°C | 無 | 梁、一般結構構材 |
| **C 級** | 有上限 | ≤ 0.80 | 27J@0°C | 有（RA≥25%） | 厚板柱、層狀撕裂風險構材 |

### 強度等級

| 強度等級 | Fy (MPa) | Fu (MPa) |
|---------|---------|---------|
| SN400A/B/C | 235~355 | 400~510 |
| SN490B/C | 325~445 | 490~610 |

### B 級的關鍵特性

1. **Fy 上下限控制**：避免鋼廠過強供料（上限）和不足（下限）
   - 目的：確保強柱弱梁計算的準確性（Fy 變異 → 強度比計算失準）

2. **降伏比 ≤ 0.80**：確保足夠的塑性變形能力

3. **CVN ≥ 27J @ 0°C**：確保不發生低溫脆裂

### C 級的額外要求（Z向韌性）

**Z 向（板厚方向）韌性**：防止層狀撕裂（Lamellar Tearing）

```
層狀撕裂（Lamellar Tearing）：
    厚板在板厚方向受拉時（如 T 型接合）
    沿板厚方向的夾雜物形成裂縫層
    → 沿厚度方向撕裂，平行板面
```

**規範要求使用 SN-C 級的情況：**
- 箱形柱板厚 ≥ 40 mm
- T 型接合（穿透型銲接）受厚度方向拉力
- 轉角板（角接）有撕裂風險

## 斷裂力學基礎概念

### 應力強度因子 K（Stress Intensity Factor）

$$K = \sigma \sqrt{\pi a} \cdot F$$

- $a$：裂縫半長
- $F$：幾何修正係數
- $K_{IC}$：平面應變斷裂韌性（材料常數）

**斷裂條件**：$K \geq K_{IC}$ → 裂縫不穩定擴展

### CVN 與 KIC 的關聯

$$K_{IC} \approx 0.64 \sqrt{CVN \cdot \sigma_y}$$

（近似換算，用於工程估算）

### 影響斷裂韌性的因素

| 因素 | 效應 |
|------|------|
| 溫度↓ | 韌性↓（最重要） |
| 板厚↑ | 韌性↓（三軸拘束效應） |
| 應變速率↑ | 韌性↓ |
| 夾雜物多 | 韌性↓ |
| 晶粒細化 | 韌性↑ |
| 含碳量↑ | 韌性↓（但強度↑） |

## 銲接熱影響區（HAZ）的脆化

銲接時 HAZ 受高溫後快速冷卻：

```
HAZ 分區（從銲道向外）：
    ① 過熱區（900~1500°C）：晶粒粗大，韌性最低
    ② 正火區（700~900°C）：細晶，韌性最好
    ③ 部分相變區（A1~A3）：混合組織
    ④ 回火區（低於 A1）：原有組織略改變
```

**HAZ 脆化的設計對策：**
- 選用低碳當量鋼材（CE ≤ 0.36 for SN400）
- 施工預熱（降低冷卻速率）
- 後熱處理（PWHT）
- 選用 SN-B/C 級鋼材（已確認足夠韌性）

## 常見考題型式

1. **說明題**：「說明 SN 鋼材 B 級與 C 級的差異及其適用場合」
   - 答：B級（CVN, Fy 雙限）vs. C級（額外Z向韌性 RA≥25%）；C級用於厚板柱（≥40mm）防層狀撕裂

2. **概念題**：「為何耐震設計要限制降伏比？」
   - 答：降伏比低 → 降伏後至斷裂有足夠應變空間 → 塑性變形能力足夠 → 耗能能力強

3. **說明題**：「層狀撕裂的發生機制與防治」
   - 答：機制（Z向拉力+夾雜物）→ 防治（使用C級鋼 RA≥25%，或改接合型式）
