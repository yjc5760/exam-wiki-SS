## 材料行為與失敗模式的關係

```
σ-ε 曲線
    ├─ Fy（降伏強度）→ 強度失敗設計值（毛斷面降伏）
    ├─ Fu（極限強度）→ 強度失敗設計值（淨斷面斷裂）
    └─ 破斷延伸率 → 韌性指標

殘留應力
    └─ 使非彈性挫屈提前（柱強度曲線下降）

斷裂韌性
    ├─ CVN 低 → 低溫脆裂風險
    └─ SN 鋼材 C 級 → Z 向韌性（厚板防層狀撕裂）

銲接性
    ├─ 碳當量高 → 需預熱
    └─ HAZ 脆化 → 焊後熱處理（PWHT）
```
