# 殘留應力（Residual Stress）

> **核心問題：** 製造過程產生的初始應力如何影響柱的非彈性挫屈強度？

---

## 定義

殘留應力（Residual Stress）：構材**在無外力作用時**就存在的自我平衡內應力，由製造過程中不均勻的冷卻、加工或焊接產生。

$$\sum F_{residual} = 0, \quad \sum M_{residual} = 0$$

（在任何截面上，殘留應力必須自我平衡）

---

## 殘留應力的來源

### 1. 熱軋 H 型鋼（最主要）

```
冷卻順序（從最快到最慢）：
    ① 翼板尖端（最先冷卻）→ 先收縮 → 周圍材料阻礙 → 拉應力
    ② 翼腹交接處（最後冷卻）→ 後收縮 → 被周圍已冷卻材料拉住 → 壓應力

等效殘留應力分布：
    翼板：兩端拉（+），中央壓（−）
    腹板：兩端壓（−），中央拉（+）
```

### 典型殘留應力大小

$$\sigma_{rc} \approx -0.3 F_y \text{（翼板端部壓應力）}$$

$$\sigma_{rt} \approx +0.3 F_y \text{（腹板中央拉應力）}$$

### 2. 焊接引起的殘留應力

- 焊道冷卻後收縮 → 焊道附近拉應力，遠離焊道處壓應力
- 組合柱（箱形柱）焊接後殘留應力更複雜

### 3. 火焰切割（Flame Cutting）

- 切割邊緣：高溫後快速冷卻 → 拉應力帶
- 影響疲勞壽命和斷裂韌性

## 殘留應力對柱強度的影響

### 機制

受壓柱的截面應力分布：

$$\sigma_{total} = \sigma_{external} + \sigma_{residual}$$

翼板尖端：$\sigma_{total} = f_{avg} + \sigma_{rc} = f_{avg} - 0.3F_y$

當翼板尖端先達 Fy：

$$f_{avg} + 0.3F_y = F_y \Rightarrow f_{avg} = 0.7F_y$$

**換言之，在平均應力僅達 0.7Fy 時，翼板尖端已開始降伏 → 有效截面縮小 → 切線模數下降。**

### 對柱強度曲線的影響

```
Fcr/Fy
  1.0 ┤塑性區（短柱）
      │    ← 殘留應力使非彈性挫屈區
  0.7 ┤    ← 提早開始（λc 較小時就降伏）
      │    非彈性挫屈區（殘留應力主宰）
      │           ↘
      │              彈性 Euler 區（λc 大，Fy 不影響）
  0   └─────────────────────────── λc
      0    0.5   1.0   1.5   2.0
```

殘留應力導致柱強度曲線比純彈性 Euler 曲線低（非彈性區尤其明顯）。

### 切線模數理論（Tangent Modulus）

$$F_{cr} = \frac{\pi^2 E_t}{(KL/r)^2}$$

- $E_t$：切線模數（降伏後有效彈性模數）
- $E_t < E$（材料部分降伏後剛度降低）
- 殘留應力 → 更大的截面降伏面積 → $E_t$ 更小

詳見：[[TANGENT-MODULUS-THEORY]]

## 殘留應力對梁的影響

### LTB（側向扭轉挫屈）

殘留應力使梁翼板壓力側提前降伏 → 切線模數降低 → 非彈性 LTB 起點（Lr 點）提前。

**設計影響**：LTB 三區段中的非彈性區邊界（Lp, Lr）即考慮殘留應力效應。

### 梁的殘留應力分布

```
H型鋼梁（受正彎矩）：
    下翼板（受拉）：
        外加拉應力 + 殘留應力（翼板端可能為拉或壓）
    上翼板（受壓）：
        外加壓應力 + 殘留壓（翼板端） → 最先達到 Fy
```

## 殘留應力的測量方法

| 方法 | 原理 | 精度 |
|------|------|------|
| 切條法 | 切出條帶，量測彈性回彈 | 好，但破壞試件 |
| X 射線繞射 | 晶格間距變化 | 高，但深度有限 |
| 中子繞射 | 穿透力更強 | 最好，成本高 |
| 磁彈效應 | 磁性隨應力變化 | 快速，精度中等 |

---

## 設計規範的處理方式

AISC 和台灣規範**隱性**地將殘留應力效應納入柱強度公式：

$$F_{cr} = \begin{cases} 0.658^{\lambda_c^2} F_y & \lambda_c \leq 1.5 \\ \dfrac{0.877}{\lambda_c^2} F_y & \lambda_c > 1.5 \end{cases}$$

- **非彈性挫屈公式（λc ≤ 1.5）**：基於切線模數理論，已含殘留應力效應
- **彈性挫屈公式（λc > 1.5）**：0.877 折減（考慮幾何缺陷，非殘留應力）

## 常見考題型式

1. **說明題**：「說明殘留應力的來源、分布及對柱強度的影響」
   - 答：來源（不均勻冷卻）→ 分布（翼板端壓、腹板中拉）→ 影響（非彈性挫屈提前，0.7Fy 時部分降伏）

2. **計算題**（間接）：柱強度公式已含殘留應力，不需額外計算

3. **繪圖題**：繪出殘留應力分布圖（翼板端 -0.3Fy，腹板中 +0.3Fy）
