# 應力-應變行為（σ-ε Curve）

> **核心問題：** 鋼材如何從彈性進入塑性？各設計強度值從哪裡來？

---

## 標準 σ-ε 曲線四個區段

```
σ (MPa)
  │                           ___________破斷點
  │                    ______/ 應變硬化區
Fu│           ________/
  │          /降伏平台（Fy 不變，ε 增加）
Fy│_________/
  │        /
  │       / 彈性區（斜率 = E）
  │      /
  │     /
  │────/─────────────────────────── ε
       εy   εst          εu    εf
```

| 區段 | 應力範圍 | 特徵 | 設計使用 |
|------|---------|------|---------|
| 彈性區 | σ < Fy | 線性，E = 200 GPa | 彈性分析 |
| 降伏平台 | σ = Fy | ε 增加，σ 不變 | LRFD/ASD 設計值 |
| 應變硬化 | σ > Fy | 斜率 = Est < E | 塑性設計上限 |
| 破斷 | σ = Fu | 頸縮後斷裂 | 淨斷面設計值 |

---

## 關鍵材料參數

### 彈性模數 E

$$E = 200{,}000 \text{ MPa} = 200 \text{ GPa} = 2040 \text{ tf/cm}^2$$

### 降伏強度 Fy 與極限強度 Fu

| 鋼材 | Fy (MPa) | Fu (MPa) | Fu/Fy |
|------|---------|---------|-------|
| A36 | 250 | 400~550 | ≥ 1.5 |
| A572 Gr.50 | 345 | 450 | 1.30 |
| A992 | 345~448 | 448~621 | ≤ 1.5（且 Fy/Fu ≤ 0.85） |
| SN400B | 235~355 | 400~510 | ≤ 0.80（上下限控制）|
| SN490B | 325~445 | 490~610 | ≤ 0.80 |

### 降伏比（Fy/Fu）

$$\text{降伏比} = \frac{F_y}{F_u}$$

- 降伏比越低 → 韌性越好（能量吸收能力強）
- 耐震鋼材要求降伏比 ≤ 0.80（SN 鋼材）或 ≤ 0.85（A992）
- 降伏比過高 → 降伏後立即斷裂，無延性

### 泊松比與剪切模數

$$\nu = 0.3 \quad G = \frac{E}{2(1+\nu)} \approx 77{,}000 \text{ MPa}$$

## 設計值的使用規則

### LRFD 設計強度

| 失敗模式 | 設計強度 | 使用強度 |
|---------|---------|---------|
| 毛斷面降伏 | $\phi_t = 0.90$ | Fy |
| 淨斷面斷裂 | $\phi_t = 0.75$ | Fu |
| 銲縫剪斷 | $\phi = 0.75$ | 0.6 FEXX |
| 螺栓剪斷 | $\phi = 0.75$ | Fnv |
| 柱整體挫屈 | $\phi_c = 0.85$ | Fcr ≤ Fy |

### 為什麼φ值不同？

| φ值 | 對應失敗型式 | 理由 |
|-----|-----------|------|
| 0.90 | 降伏（延性） | 降伏有預警，散布較小 |
| 0.85 | 挫屈（穩定） | 幾何缺陷影響大，不確定性較高 |
| 0.75 | 斷裂（脆性） | 脆性破壞無預警，需更高安全裕度 |

## 設計假設 vs. 實際行為

### 假設 1：理想彈塑性（Elastic-Perfectly Plastic）

大部分設計採用：

```
σ
Fy ─────────────
   /
  / 彈性
  /
  ───────────── ε
```

**忽略應變硬化**（保守設計）。

### 假設 2：殘留應力的影響

實際 σ-ε 曲線中，**殘留應力**使降伏提前在部分纖維發生：
- H 型鋼翼板尖端殘留壓應力 ≈ 0.3Fy
- 受壓時，翼板尖端 σ = 外加壓 + 殘留壓，先行降伏
- 有效斷面縮小 → 切線模數下降

詳見：[[residual-stress]]

## 超強係數（Ry, Rt）

耐震設計中，實際 Fy 可能高於標稱 Fy（鋼廠過強供料）：

$$R_y = \frac{F_{y,expected}}{F_{y,nominal}}$$

| 鋼材 | Ry |
|------|-----|
| A36 | 1.5 |
| A992 | 1.1 |
| SN490 | 1.1 |

**用途**：容量設計時，強構材需能承受弱構材超強後的力。

$$M_{pr} = R_y F_y Z_x \times 1.1 \text{（或其他規定放大係數）}$$
