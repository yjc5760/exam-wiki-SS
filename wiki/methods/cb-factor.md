# 彎矩梯度修正因子 $C_b$（Moment Gradient Modification Factor）

**方法 ID：** cb-factor
**知識分類：** 解題工具 / LTB 強度修正
**適用題型：** SS-U1-2 梁桿件、SS-U1-3 梁柱桿件

---

## 核心原理

**$C_b$（彎矩梯度修正因子）** 量化「非均勻彎矩分布對 LTB 強度的提升效果」。

LTB 臨界強度公式是針對**均勻彎矩**推導的，而實際梁的彎矩圖往往是非均勻的（三角形、拋物線、反曲線等）。非均勻彎矩下：

- 只有少數截面承受最大彎矩 $M_{max}$
- 彎矩較小的區段能「拉住」最大彎矩處，抑制 LTB 發展
- 相當於縮短了有效挫屈長度

$C_b > 1.0$ 表示此非均勻彎矩的 LTB 強度比同長度均勻彎矩**提高了 $C_b$ 倍**。

> **直覺理解：** 均勻彎矩（$C_b = 1.0$）整段都處於最不利狀態；三角形彎矩（$C_b \approx 1.67$）只有一端受最大彎矩，另一端接近零，較有利。

---

## 設計公式

### AISC 四分點公式（LRFD 標準）

$$\boxed{C_b = \frac{12.5\,M_{max}}{2.5\,M_{max} + 3M_A + 4M_B + 3M_C} \leq 3.0}$$

| 符號 | 定義 | 注意 |
|------|------|------|
| $M_{max}$ | 無支撐段內最大彎矩絕對值 | 永遠取**絕對值** |
| $M_A$ | 無支撐段 $1/4$ 點彎矩絕對值 | 永遠取**絕對值** |
| $M_B$ | 無支撐段 $1/2$ 點（中點）彎矩絕對值 | 永遠取**絕對值** |
| $M_C$ | 無支撐段 $3/4$ 點彎矩絕對值 | 永遠取**絕對值** |

> ⚠️ **所有彎矩均取絕對值**，包含有正負彎矩混合的反曲線情況。

### 在 LTB 三區段的應用

$$M_n = C_b \times M_{n,\,C_b=1.0} \leq M_p$$

| LTB 區間 | 加入 $C_b$ 後的公式 |
|---------|-------------------|
| 塑性區（$L_b \leq L_p$） | **$C_b$ 不適用此區間**，$M_n = M_p$ |
| 非彈性 LTB（$L_p < L_b \leq L_r$） | $M_n = C_b \!\left[M_p - (M_p - M_r)\dfrac{L_b - L_p}{L_r - L_p}\right] \leq M_p$ |
| 彈性 LTB（$L_b > L_r$） | $M_n = C_b\,M_{cr} \leq M_p$ |

> 💡 **關鍵上限：** 加入 $C_b$ 後，$M_n$ 不得超過 $M_p$（截面塑性彎矩為絕對上限）。

## 邊界條件 / 常見情況對照表

| 彎矩分布類型 | 描述 | $C_b$ 值 | 備註 |
|------------|------|---------|------|
| 均勻彎矩 | 水平線 | **1.0** | 最不利，LTB 規範基準 |
| 線性（三角形） | 一端最大，一端為零 | **1.67** | 如端部集中彎矩 |
| 拋物線（均布載重） | 中點最大 | **1.14** | 簡支梁均布載重 |
| 對稱三角形（跨中集中載重） | 中點最大，線性至兩端 | **1.35** | 簡支梁跨中 $P$ |
| 反曲線（雙曲率） | S 形，正負彎矩混合 | **≈ 2.3** | 最有利，樓架梁常見 |
| 懸臂梁（自由端無束制） | 固端最大 | **1.0\*** | \*AISC 規定直接取 1.0 |

> ⚠️ **懸臂梁特殊規定：** 自由端無側向束制時，AISC 規定直接取 $C_b = 1.0$，不套用四分點公式。

## 計算步驟流程

```
Step 1  確認無支撐段範圍（側向支撐點位置）
        Lb = 相鄰側向支撐點間距

Step 2  繪製（或讀取）彎矩圖，找出控制段（Mmax/Mn 比值最大的段）

Step 3  取控制段四個關鍵點的彎矩絕對值

        支撐點A ──┬──────────┬──────────┬──────────┬── 支撐點B
                  Lb/4      Lb/2      3Lb/4
                  MA        MB        MC      （Mmax 在段內任意位置）

Step 4  代入公式計算 Cb

Step 5  確認上限 Cb ≤ 3.0

Step 6  確認 Lb 所在 LTB 區間，代入對應公式
        計算 Mn = Cb × Mn(Cb=1.0)，並與 Mp 取小值
```

## 推導示範

### 範例一：線性（三角形）彎矩分布

**設定：** 簡支梁跨中設側向支撐，分析端部集中彎矩造成的三角形彎矩段（$L_b = 4.5$ m，$M_{max} = 42$ kN·m，另一端彎矩為 0）

線性插值：
- $M_A = 42 \times \frac{1}{4} = 10.5$ kN·m
- $M_B = 42 \times \frac{2}{4} = 21.0$ kN·m
- $M_C = 42 \times \frac{3}{4} = 31.5$ kN·m

$$C_b = \frac{12.5 \times 42}{2.5 \times 42 + 3 \times 10.5 + 4 \times 21 + 3 \times 31.5} = \frac{525}{105 + 31.5 + 84 + 94.5} = \frac{525}{315} = \mathbf{1.667}$$

> 💡 **驗證：** 線性（三角形）彎矩的理論 $C_b = 5/3 \approx 1.667$，完全吻合。

---

### 範例二：跨中集中載重（對稱三角形）

**設定：** 簡支梁跨中集中載重 $P$，側向支撐在兩端 + 跨中（$C$ 點）。分析右半段（$C$–$B$，彎矩從 $M_{max}$ 線性降至 0）。令 $M_{max} = 42$ kN·m。

從支撐點 $C$ 量起（$C$ 端彎矩最大，$B$ 端為零）：
- $M_A = 42 \times \frac{3}{4} = 31.5$ kN·m
- $M_B = 42 \times \frac{2}{4} = 21.0$ kN·m
- $M_C = 42 \times \frac{1}{4} = 10.5$ kN·m

$$C_b = \frac{12.5 \times 42}{2.5 \times 42 + 3 \times 31.5 + 4 \times 21 + 3 \times 10.5} = \frac{525}{315} = \mathbf{1.667}$$

> 💡 兩種三角形（從零到最大、或從最大到零）結果相同，均為 1.667。

## 考場安全答法

**Step 1（30 秒）：** 判斷是否需要計算 $C_b$
- 題目給定 $C_b = 1.0$ → 直接使用
- 題目說「均勻彎矩」→ $C_b = 1.0$
- 題目給出彎矩圖 → 必須用公式計算

**Step 2（90 秒）：** 讀取四個關鍵點彎矩（取絕對值）

> ⚠️ **最常見陷阱：忘記取絕對值**。反曲線段有正有負，$M_A$、$M_B$、$M_C$ 全部取絕對值。

**Step 3（30 秒）：** 代入公式，記住上限 $C_b \leq 3.0$

**Step 4（30 秒）：** 乘入 LTB 公式，記住上限 $M_n \leq M_p$

$$\boxed{C_b = \frac{12.5\,M_{max}}{2.5\,M_{max} + 3M_A + 4M_B + 3M_C} \leq 3.0 \quad\Rightarrow\quad M_n = C_b \times M_{n,1.0} \leq M_p}$$

## 與其他方法的比較

| 情境 | 做法 |
|------|------|
| 題目直接給 $C_b$ | 直接代入，不需計算 |
| 均勻彎矩（純彎） | $C_b = 1.0$，最保守 |
| 有彎矩圖且非均勻 | 用四分點公式計算，能提升強度 |
| 懸臂梁自由端無束制 | AISC 規定 $C_b = 1.0$，不套公式 |
