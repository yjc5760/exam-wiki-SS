# 偏心銲縫群（彈性向量法）

**方法 ID：** eccentric-weld
**知識分類：** 解題工具 / 接合設計
**適用題型：** SS-U1-4 接合設計（偏心銲縫接合）
**出現題數：** 3 題

---

## 核心原理

偏心銲縫接合與偏心螺栓群原理相同，但銲縫是**連續分佈**，需將銲縫視為面積線（Line Element），以**單位長度**表示力。

彈性向量法：
- 將銲縫群視為線圖（每段銲縫寬度取 1，計算有效喉深後乘以）
- 計算銲縫群形心和極慣性矩 $I_p$（線圖的極慣性矩，單位 cm³）
- 找出最大應力點（最大合力點），與銲縫容許力比較

設計強度：
- LRFD：$\phi R_n = 0.75 \times 0.6 F_{EXX} \times 0.707w \times L_{weld}$
- ASD：$F_w = 0.3 F_{EXX}$（容許應力）

## 解題步驟

```
Step 1  畫出銲縫配置（銲縫線圖）
        建立座標系，以銲縫群形心為原點

Step 2  計算銲縫群形心（各段銲縫長度加權）
        x̄ = Σ(Li × xi,centroid) / ΣLi
        ȳ = Σ(Li × yi,centroid) / ΣLi

Step 3  計算銲縫群的極慣性矩 Ip（線圖）
        水平銲縫段（長 b，形心在 (xo, yo)）：
            Ix,局部 = b³/12 + b×yo²  （對整體形心）
            Iy,局部 = 0 + b×xo²      （水平段無 Iy,局部）
        垂直銲縫段（長 d，形心在 (xo, yo)）：
            Ix,局部 = 0 + d×yo²
            Iy,局部 = d³/12 + d×xo²
        Ip = ΣIx,局部 + ΣIy,局部（所有銲縫段）

Step 4  計算直接剪力（單位銲縫長度）
        fv = P / ΣL（所有銲縫總長）
        水平分量 fvx，垂直分量 fvy

Step 5  計算扭矩引起的銲縫力（最大應力點）
        M = P × e
        對最遠點（最大 r = √(x² + y²)）：
        fMx = M × y / Ip（水平方向分量）
        fMy = M × x / Ip（垂直方向分量）
        方向垂直於 r，與扭矩方向一致

Step 6  向量疊加求合力（單位長度）
        fx = fvx + fMx
        fy = fvy + fMy
        f = √(fx² + fy²)

Step 7  換算為實際銲道力
        Runit = f × 1（每單位長度）
        φRn_unit = 0.75 × 0.6 × FEXX × 0.707w

Step 8  驗核：f ≤ φRn_unit → 決定所需銲道尺寸 w
```

## 常用銲縫線圖 Ip 公式速查

**C 形銲縫（兩垂直段 + 一水平段）：**

假設外框尺寸 $b$（寬）× $d$（高），各段厚均為線圖（寬 = 1）：

| 形狀 | Ip（相對自身形心） |
|------|-----------------|
| 水平段長 $b$ | $b^3/12 + b \cdot \bar{y}^2$ |
| 垂直段長 $d$ | $d^3/12 + d \cdot \bar{x}^2$ |

**矩形銲縫（四周均有銲縫）：**

$$I_p = \frac{b(3d^2 + b^2)}{6} + \frac{d(3b^2 + d^2)}{6}$$

實務上查 AISC 銲縫群表（Table 8-42 等）。

## 常見陷阱

| # | 陷阱 | 對策 |
|---|------|------|
| 1 | **Ip 是「線圖」極慣性矩，單位 cm³** | 不能直接用面積公式，要用線圖公式（每段 $bd^3/12$ 的變形版） |
| 2 | **形心計算錯誤** | 各銲縫段用其長度加權，不是面積加權 |
| 3 | **最大應力點不一定是最遠點** | 要同時考慮方向，向量合力最大的點才是控制點 |
| 4 | **直接剪力用總銲縫長** | $f_v = P / \Sigma L$，不是 $P / $ 一段銲縫長度 |
| 5 | **φ = 0.75，FEXX = 70 ksi（E70XX）** | 台灣題目常用 E70XX，$F_{EXX} = 49$ tf/cm² |
