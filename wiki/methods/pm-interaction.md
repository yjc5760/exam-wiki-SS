# P-M 互制驗核流程（梁柱桿件）

**方法 ID：** pm-interaction
**知識分類：** 解題工具 / 梁柱桿件設計
**適用題型：** 4.1.3 梁柱桿件
**出現題數：** 6 題（梁柱題幾乎必包含）

---

## 核心原理

梁柱同時承受軸力 $P_u$ 與彎矩 $M_u$，兩者共同消耗斷面強度，以**兩段線性互制方程式**描述安全邊界：

$$\text{當 } \frac{P_u}{\phi_c P_n} \geq 0.2：\quad \frac{P_u}{\phi_c P_n} + \frac{8}{9}\left(\frac{M_{ux}}{\phi_b M_{nx}} + \frac{M_{uy}}{\phi_b M_{ny}}\right) \leq 1.0 \quad \cdots (H1\text{-}1a)$$

$$\text{當 } \frac{P_u}{\phi_c P_n} < 0.2：\quad \frac{P_u}{2\phi_c P_n} + \left(\frac{M_{ux}}{\phi_b M_{nx}} + \frac{M_{uy}}{\phi_b M_{ny}}\right) \leq 1.0 \quad \cdots (H1\text{-}1b)$$

- $\phi_c P_n = 0.85 F_{cr} A$（壓力強度，含 $\lambda_c$ 計算）
- $\phi_b M_n = 0.9 M_n$（彎矩強度，含 LTB 判斷）
- $M_u$ 須為**二階彎矩**（含 $B_1$、$B_2$ 放大，見 B1B2 Method）

---

## 解題步驟

```
Step 1  計算軸壓設計強度 φcPn
        → 用 LRFD-Column 流程（λc → Fcr → φcPn）

Step 2  計算彎矩設計強度 φbMnx（及 φbMny 若有雙軸彎矩）
        → 用 LTB-3Zone 流程（確認 Lb → Mn → φbMn）
        單軸彎矩只算 x 軸；雙軸彎矩兩軸都算

Step 3  計算二階設計彎矩 Mux（含 B1B2 放大）
        有側移框架：Mux = B1 Mnt + B2 Mlt
        無側移構件：Mux = B1 Mnt

Step 4  判斷適用方程式
        計算 Pu/(φcPn)：
        ≥ 0.2 → 用 H1-1a
        < 0.2 → 用 H1-1b

Step 5  代入互制方程式，確認 ≤ 1.0
        若 > 1.0 → 斷面不足，需加大斷面
        若遠 < 1.0 → 斷面超大，可縮小（斷面選擇題）

Step 6  P-M 互制圖繪製（若題目要求）
        x 軸：M/φbMn（0 至 1）
        y 軸：P/φcPn（0 至 1）
        折線：(0, 1) → (1/9, 0.2) → (1, 0)（H1-1a/b 的兩段）
        標示設計點 (Mux/φbMnx, Pu/φcPn) 並確認在折線內
```

---

## ASD 梁柱互制方程式

$$\frac{f_a}{F_a} + \frac{C_m f_{bx}}{(1 - f_a/F'_{ex})F_{bx}} + \frac{f_{by}}{(1-f_a/F'_{ey})F_{by}} \leq 1.0$$

$$\frac{f_a}{0.6F_y} + \frac{f_{bx}}{F_{bx}} + \frac{f_{by}}{F_{by}} \leq 1.0$$

取兩個方程式均需滿足（同 LRFD 取嚴者控制）。

## 常見陷阱

| # | 陷阱 | 對策 |
|---|------|------|
| 1 | **先判斷 Pu/φcPn 再選公式** | 不要直接套 H1-1a，0.2 是分水嶺 |
| 2 | **φc ≠ φb**：壓力 0.85，彎矩 0.90 | 壓力桿件 φc=0.85；彎矩桿件 φb=0.90 |
| 3 | **Mux 要用二階彎矩** | 題目給的是一階彎矩時，需先乘 B1（構件彎矩）和 B2（層間彎矩） |
| 4 | **雙軸彎矩兩項都要加** | 有 Muy 時，式中第二項是 Mux/φbMnx + Muy/φbMny |
| 5 | **P-M 圖折點在 Pu/φcPn = 0.2** | 折線不是直線，在 0.2 處有轉折 |
| 6 | **強柱弱梁中 Mpc 含軸壓修正** | $M_{pc} = Z_x(F_y - P_u/A_g)$，不是 $Z_x F_y$ |
