# ASD 插銷鉸接拉力構件設計哲學（SS-U1-1）

**哲學 ID：** asd-pin-tension
**適用規範：** 容許應力設計法（ASD）
**命題大綱：** SS-U1-1 拉力及壓力桿件

---

## 設計邏輯

```
設計服務載重：P = PD + PL（不因數化）

四種破壞模式（依序檢核）：

① 全斷面降伏
   Pn = Fy × Ag，Ω = 1.67
   Pa = Pn / Ω ≥ P

② 有效淨面積斷裂（Pin 孔邊）
   be = min(2t + 16, 實際)  ← be 單位 mm
   Pn = Fu × (2t × be)，Ω = 2.00

③ 剪力斷裂（剪出破壞）
   Asf = 2t × (a + d/2)  ← a = 孔邊至端部淨距，d = 孔徑
   Pn = 0.6Fu × Asf，Ω = 2.00

④ 承壓
   Apb = t × d
   Pn = 1.8Fy × Apb，Ω = 2.00

取最小 Pa = min(各 Pn/Ω)，確認 Pa ≥ P
```

## 常見陷阱

1. **be 的上限**：$b_e = \min(2t+16,\text{實際寬})$，單位 mm，常代錯
2. **剪出面積**：$A_{sf} = 2t(a + d/2)$，兩側面積，有 2 倍
3. **承壓投影面積**：$A_{pb} = t \times d$（孔徑 × 板厚），而非接觸面積
