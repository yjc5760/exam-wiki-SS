# LRFD 梁柱桿件設計哲學（箱形斷面，SS-U1-3）

**哲學 ID：** lrfd-beam-column-box
**適用規範：** 鋼結構極限設計法（LRFD）
**命題大綱：** SS-U1-3 梁柱桿件

## 箱形斷面性質（正方形 BOX $b_o \times b_o \times t$）

$$b_i = b_o - 2t$$

| 性質 | 公式 |
|------|------|
| 面積 | $A = b_o^2 - b_i^2$ |
| 慣性矩 | $I = (b_o^4 - b_i^4)/12$ |
| 彈性斷面模數 | $S = I / (b_o/2)$ |
| 塑性斷面模數 | $Z = (b_o^3 - b_i^3)/4$ |
| 迴轉半徑 | $r = \sqrt{I/A}$ |

## 設計邏輯

```
① 局部挫屈檢核
   b/t = bi/t ≤ 50/√Fy（Fy 單位 tf/cm²）

② 壓力強度 φcPn
   計算 λc → Fcr → Pn

③ 彎矩強度 φbMn
   充分側向支撐 → 無 LTB → φbMn = 0.9 FyZ

④ 剪力強度
   受 V 方向的兩片平行腹板：Aw = 2 × bi × t
   h/tw ≤ 50√(kv/Fy) → Vn = 0.6FyAw

⑤ P-M 互制檢核
   Pu/(φcPn) ≥ 0.2 → H1-1a：
     Pu/(φcPn) + (8/9)[Muy/φbMny + Muz/φbMnz] ≤ 1.0

   Pu/(φcPn) < 0.2 → H1-1b：
     Pu/(2φcPn) + [Muy/φbMny + Muz/φbMnz] ≤ 1.0
```

## 常見陷阱

1. **剪力面積**：箱形柱受剪方向有兩片腹板，$A_w = 2 h t$
2. **$b_i$ 的定義**：清淨內寬，非外寬
3. **互制判斷門檻**：$P_u/\phi_c P_n$ 與 0.2 比較決定套哪條公式
