# LRFD 梁桿件設計哲學（4.1.2）

**哲學 ID：** lrfd-beam
**適用規範：** 鋼結構極限設計法（LRFD）
**命題大綱：** 4.1.2 梁桿件
**核心原則：** $\phi_b M_n$ 取所有極限狀態中最脆弱一環

---

## 設計邏輯

```
① 基準強度（全斷面塑性化）
   Mp = Fy × Zx

② 局部挫屈檢核（寬厚比 λ）
   λ ≤ λp → 結實斷面（可達 Mp）
   λp < λ ≤ λr → 非結實（線性折減）
   λ > λr → 細薄斷面（彈性挫屈控制）

③ 側向扭轉挫屈 LTB（比較 Lb 與 Lp、Lr）
   Lb ≤ Lp → 塑性區，φbMn = φbMp
   Lp < Lb ≤ Lr → 非彈性 LTB（線性內插，乘以 Cb）
   Lb > Lr → 彈性 LTB（Mcr 公式，乘以 Cb）
   上限：Mn ≤ Mp

④ 剪力強度檢核（h/tw 三段式）
   h/tw ≤ 50√(kv/Fyw) → Vn = 0.6FywAw
   超出後依序折減

⑤ 使用性撓度（未因數化活載重）
   δL ≤ L/360（一般活載重）
```

## 考場策略

- **斷面選擇題**：先用 $\phi_b M_p$ 篩選，再確認 $L_b$ 與 $L_p$
- **LTB 計算**：先判斷區間，再套公式；$C_b$ 在非彈性與彈性區均有效
- **剪力常被忽略**：有集中載重或短跨梁時，剪力可能控制
