## 題幹摘要

**題幹：** 有一鋼造斜撐構架系統，經過載重組合分析後得柱軸力 800 kN，柱底端強軸彎矩為 100 kN·m，弱軸彎矩為 50 kN·m，柱底端為固接端，另一端為鉸接端，此柱在強軸及弱軸的**有效長度（KL）均為 4.5 m**，請利用極限設計法（LRFD）**檢核此柱所承受之軸力彎矩需求強度比**是否符合設計需求。（25 分）

**給定條件：**

| 項目 | 數值 |
|------|------|
| 鋼材 | ASTM A572 Gr.50 |
| $F_y$ | 350 MPa |
| $E$ | 200,000 MPa |
| $P_u$（軸壓力，已因數化） | 800 kN |
| $M_{ux}$（強軸彎矩，已因數化） | 100 kN·m（柱底固接端） |
| $M_{uy}$（弱軸彎矩，已因數化） | 50 kN·m（柱底固接端） |
| 柱底端條件 | 固接（有彎矩） |
| 柱頂端條件 | 鉸接（無彎矩） |
| 有效長度 $KL$（強弱軸） | 4.5 m |

**W14×68 斷面性質（題目提供）：**

| 性質 | 數值 |
|------|------|
| $Z_x$（強軸塑性斷面模數） | 1,884,512 mm³ |
| $Z_y$（弱軸塑性斷面模數） | 604,683 mm³ |
| $A$（斷面積） | 12,903 mm² |
| $r_x$（強軸迴旋半徑） | 153 mm |
| $r_y$（弱軸迴旋半徑） | 62 mm |

**規範公式（題目提供）：**

$$\lambda_c = \frac{KL}{r\pi}\sqrt{\frac{F_y}{E}}; \quad F_{cr} = (0.658)^{\lambda_c^2} F_y \ (\lambda_c \leq 1.5); \quad F_{cr} = \frac{0.877}{\lambda_c^2} F_y \ (\lambda_c > 1.5)$$

$$B_1 = \frac{C_m}{1 - P_u/P_{el}}$$

$$\frac{P_u}{2\phi P_n} + \left(\frac{M_{ux}}{\phi M_{nx}} + \frac{M_{uy}}{\phi M_{ny}}\right), \quad P_u < 0.2\phi P_n$$

$$\frac{P_u}{\phi P_n} + \frac{8}{9}\left(\frac{M_{ux}}{\phi M_{nx}} + \frac{M_{uy}}{\phi M_{ny}}\right), \quad P_u \geq 0.2\phi P_n$$

## 核心考點

**核心觀念：** 本題考驗梁柱（Beam-Column）的完整 LRFD 檢核流程，包含：
1. 以弱軸（控制軸）計算柱強度 $\phi_c P_n$
2. 判斷是否需要 $B_1$ 力矩放大（斜撐構架，無側移，$B_2 = 1$）
3. 依 $P_u/\phi_c P_n$ 比值選擇正確的 P-M 互制公式

**特別之處：**
- 柱為固接-鉸接（底固頂鉸），單曲率，彎矩梯度 $C_m = 0.6$
- $B_1$ 計算結果 $< 1.0$，需取 $B_1 = 1.0$（不能縮減一階彎矩）
- 雙軸彎矩同時作用，必須代入互制方程

## 解題關鍵步驟

**作戰計畫：**
```
Step 1: 計算 λc（弱軸控制），求 Fcr → φcPn
Step 2: 計算各軸 Pel，求 B1 → 放大彎矩（B1<1.0 取 1.0）
Step 3: 計算 φbMnx、φbMny（假設全側撐，φbMn = φbMp）
Step 4: 判斷 Pu/φcPn 比值 → 選擇 H1-1a 或 H1-1b
Step 5: 代入 P-M 互制公式，輸出 D/C 比值
```

**三大陷阱：**

1. **⚠️ 陷阱 1：控制軸判斷** — 強弱軸 KL 相同，但弱軸 $r_y = 62$ mm << 強軸 $r_x = 153$ mm，弱軸細長比大得多，**弱軸挫屈控制**。

2. **⚠️ 陷阱 2：B1 < 1.0 的處理** — 本例 $C_m = 0.6$，算出的 $B_1 < 1$，必須**取 $B_1 = 1.0$**，不可因此縮減彎矩設計值。

3. **⚠️ 陷阱 3：P-M 公式選擇** — 必須先算 $P_u/\phi_c P_n$：若 $\geq 0.2$ 用 H1-1a（含 8/9 係數），若 $< 0.2$ 用 H1-1b（除以 2 的弱化版）。

---

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：計算柱軸壓強度 φcPn

**比較各軸細長比：**

$$\frac{KL_x}{r_x} = \frac{4500}{153} = 29.41$$

$$\frac{KL_y}{r_y} = \frac{4500}{62} = 72.58 \quad \leftarrow \text{控制（較大）}$$

**計算 λc（弱軸控制）：**

$$\lambda_c = \frac{KL}{r_y \pi}\sqrt{\frac{F_y}{E}} = \frac{72.58}{\pi}\sqrt{\frac{350}{200{,}000}}$$

$$= \frac{72.58}{3.14159} \times \sqrt{0.00175} = 23.10 \times 0.04183 = \boxed{0.966}$$

$$\because \lambda_c = 0.966 < 1.5 \quad \rightarrow \text{非彈性挫屈}$$

**計算 Fcr：**

$$F_{cr} = (0.658)^{\lambda_c^2} \cdot F_y = (0.658)^{0.966^2} \cdot 350 = (0.658)^{0.933} \cdot 350$$
