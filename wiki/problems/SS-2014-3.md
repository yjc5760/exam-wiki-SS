# SS-2014-3 — W21×93 雙向彎矩梁 LRFD 檢核（非彈性LTB + 弱軸上限陷阱）

**來源：** 專門職業及技術人員高等考試結構工程技師 · 鋼結構設計 · 第三題
**考年：** 2014（民國 103 年）
**主分類：** [[SS-U1-2]] 梁桿件
**副分類：** 無
**設計法：** LRFD
**標籤：** `梁桿件` `LTB側扭挫屈` `非彈性LTB` `Cb係數` `雙向彎曲` `弱軸彎曲` `雙向彎曲互制` `結實斷面`
**驗證狀態：** ✅ verified

---

## 題幹摘要

ASTM A572 Gr.50 之 **W21×93 簡支鋼梁**，梁長 4 m，兩端均有側向支撐，梁全長承受均佈載重造成的雙向彎矩：$M_{ux} = 400$ kN-m（強軸）、$M_{uy} = 50$ kN-m（弱軸）。利用 LRFD 檢核此梁是否滿足設計要求。（25分）

給定：$F_y = 350$ MPa，$F_r = 78$ MPa，$S_x = 3{,}146{,}316$ mm³，$S_y = 362{,}154$ mm³，$Z_x = 3{,}621{,}541$ mm³，$Z_y = 568{,}631$ mm³，$L_p = 2$ m，$L_r = 5.8$ m，翼板 $\lambda_p = 9.19$

---

## 核心考點

1. **非彈性 LTB**：$L_p = 2 < L_b = 4 < L_r = 5.8$ m，線性內插公式
2. **Cb 計算**：均布載重下拋物線彎矩圖，$C_b = 12.5M_{max}/(2.5M_{max}+3M_A+4M_B+3M_C) = 1.136$
3. **弱軸強度上限**：$Z_y/S_y = 1.57 > 1.5$，必須用 $M_{ny} = 1.5F_yS_y$ 而非 $F_yZ_y$（**最重要陷阱**）
4. **雙向彎矩互制**：$M_{ux}/\phi_b M_{nx} + M_{uy}/\phi_b M_{ny} \leq 1.0$

---

## 解題關鍵步驟

```
Step 1：局部挫屈確認
        翼板 λ = bf/(2tf) = 4.53 < λp = 9.19 → 結實斷面 ✓
        腹板 h/tw = 34.1 < 89.8 → 結實斷面 ✓

Step 2：強軸 φbMnx
        Lb = 4m（無中間側撐，Lp < Lb < Lr → 非彈性 LTB）
        Cb = 12.5×400 / (2.5×400 + 3×300 + 4×400 + 3×300) = 1.136
        Mp = FyZx = 350 × 3,621,541 = 1,267.5 kN-m
        Mr = (Fy-Fr)Sx = 272 × 3,146,316 = 855.8 kN-m
        Mn = 1.136×[1267.5 - 411.7×(2/3.8)] = 1193.7 kN-m < Mp ✓
        φbMnx = 0.9 × 1193.7 = 1074.3 kN-m

Step 3：弱軸 φbMny
        Zy/Sy = 568,631/362,154 = 1.57 > 1.5 → 以上限值控制
        Mny = 1.5 × Fy × Sy = 1.5 × 350 × 362,154 = 190.1 kN-m
        φbMny = 0.9 × 190.1 = 171.1 kN-m

Step 4：雙向互制
        400/1074.3 + 50/171.1 = 0.372 + 0.292 = 0.664 ≤ 1.0 ✅
```

---

## 用到的公式

**非彈性 LTB 標稱強度：**
$$M_n = C_b \left[M_p - (M_p - M_r)\frac{L_b - L_p}{L_r - L_p}\right] \leq M_p$$

**Cb 四分點法：**
$$C_b = \frac{12.5 M_{max}}{2.5M_{max} + 3M_A + 4M_B + 3M_C} = 1.136 \text{（均布載重）}$$

**弱軸強度上限（雙對稱 I 型斷面）：**
$$M_{ny} \leq 1.5 F_y S_y \quad \text{（當 } Z_y/S_y > 1.5 \text{ 時控制）}$$

**雙向彎矩互制：**
$$\frac{M_{ux}}{\phi_b M_{nx}} + \frac{M_{uy}}{\phi_b M_{ny}} \leq 1.0$$

---

## 涉及陷阱

1. **弱軸強度上限**：$Z_y/S_y = 1.57 > 1.5$，若直接用 $F_yZ_y$ 會高估弱軸強度（扣分項） → 詳見 [陷阱：弱軸彎曲強度上限](../traps/WEAK-AXIS-BENDING-LIMIT.md)
2. **Cb ≠ 1.0**：均布載重下彎矩圖為拋物線，$C_b = 1.136 > 1.0$，若誤取 1.0 則 $\phi_b M_{nx}$ 偏保守
3. **無中間側撐則 Lb = 全梁長**：本題 $L_b = 4$ m，不是 2 m
4. **Mr 用 (Fy - Fr)**：殘留應力 Fr = 78 MPa 需代入 $M_r = (F_y - F_r)S_x$
