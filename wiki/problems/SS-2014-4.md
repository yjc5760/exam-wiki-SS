# SS-2014-4 — W14×90 鋼柱梁柱桿件 LRFD 檢核（B1/B2 二階效應 + P-M 互制）

**來源：** 專技高考結構工程技師 · 鋼結構設計 · 第四題
**考年：** 2014（民國 103 年）
**主分類：** [[4.1.3]] 梁柱桿件
**副分類：** [[6.3.1]] 結構耐震設計
**設計法：** LRFD
**標籤：** `梁柱桿件` `P-M互制` `B1放大法` `B2放大法` `二階效應` `斜撐構架` `地震力組合` `非側移框架` `容量設計`
**驗證狀態：** ✅ verified

---

## 題幹摘要

鋼造斜撐構架系統，W14×90（ASTM A572 Gr.50，$F_y = 350$ MPa）結實斷面鋼柱，柱長 3.75 m。
給定強弱軸有效長度係數（$K_x = 1.79$，$K_y = 1.0$）、垂直力分析結果（1.2D+0.5L）及地震力分析結果（1.0E），以及樓層側向參數（$\sum P_u = 9000$ kN，$H = 600$ kN，$\Delta/L = 5/1000$），以 AISC-LRFD 檢核柱是否滿足設計要求。

---

## 核心考點

1. **B2 放大係數**：處理 P-Δ 效應（樓層側移），以樓層參數計算
2. **B1 放大係數**：處理 P-δ 效應（構件自身撓曲），Pe 用非側移條件（K=1）
3. **二階彎矩**：$M_{ux} = B_1 M_{nt} + B_2 M_{lt}$（垂直力 Mnt 與地震力 Mlt 分離放大）
4. **柱軸壓強度**：計算 $\lambda_c$ → $F_{cr}$ → $\phi_c P_n$（強軸控制）
5. **P-M 互制檢核**：判斷公式（Pu/φcPn = 0.463 ≥ 0.2 → 用公式一）

---

## 解題關鍵步驟

```
Step 1：載重組合（1.2D+0.5L+1.0E）→ Pu = 2050 kN
Step 2：B2 計算（P-Δ 效應）→ B2 = 1.081
Step 3：B1 計算（P-δ 效應）→ Cm = 0.855，Pe = 58,366 kN，B1 = 0.886 → 取 1.0
Step 4：二階彎矩 → Mux = 1.0×110 + 1.081×200 = 326.2 kN-m（頂端控制）
Step 5：柱軸壓強度 → (KL/r)gov = 43.03（強軸），λc = 0.573，Fcr = 305.0 MPa，φcPn = 4433 kN
Step 6：彎矩強度 → L < Lp，無LTB，φbMnx = 0.9×FyZx = 810.4 kN-m
Step 7：P-M 互制 → 0.463 + (8/9)×0.4025 = 0.821 ≤ 1.0 ✅
```

**結論：** W14×90 柱 P-M 互制比 = **0.821 ≤ 1.0**，滿足 AISC-LRFD 設計要求。

---

## 用到的公式

$$B_2 = \frac{1}{1 - \dfrac{\sum P_u \cdot \Delta}{\sum H \cdot L}}$$

$$B_1 = \frac{C_m}{1 - P_u/P_e} \geq 1.0 \quad,\quad P_e = \frac{\pi^2 EI}{L^2} \text{（B1用K=1）}$$

$$C_m = 0.6 - 0.4\left(\frac{M_1}{M_2}\right)$$

$$M_{ux} = B_1 M_{nt} + B_2 M_{lt}$$

$$\lambda_c = \frac{KL/r}{\pi}\sqrt{\frac{F_y}{E}} \quad,\quad F_{cr} = 0.658^{\lambda_c^2} F_y \quad (\lambda_c \leq 1.5)$$

$$\text{公式一（}P_u/\phi_c P_n \geq 0.2\text{）：}\quad \frac{P_u}{\phi_c P_n} + \frac{8}{9}\left(\frac{M_{ux}}{\phi_b M_{nx}}\right) \leq 1.0$$

---

## 涉及陷阱

| 陷阱 | 說明 |
|------|------|
| ⚠️ **B2 用樓層總 ΣPu** | B2 分子的 ΣPu = 9000 kN 是**樓層所有柱**的垂直載重總和，不是本柱 Pu = 2050 kN |
| ⚠️ **B1 中 Pe 的 K 值** | B1 計算的 Pe 用 K=1.0（非側移條件），不是強軸有效長度 Kx=1.79；整體側移由 B2 處理 |
| ⚠️ **地震力載重組合** | 垂直力與地震力的組合為 1.2D+0.5L+1.0E，不是 1.2D+1.6L+1.0E |
| ⚠️ **B1 下限 = 1.0** | 計算得 B1=0.886 < 1.0，規範規定取下限 1.0，不可直接用 0.886 放大 |
