## 題幹摘要

**題目：** 鋼構架中有一柱長 400 cm，採用板厚 3.2 cm、寬度 80 cm 之箱型斷面。分析後得知柱需承受壓力 1,119 tf，且 2 方向有效長度係數分別為 2.284 與 2.397。鋼材降伏強度 3.3 tf/cm²，而揚氏係數 2,040 tf/cm²。試依極限設計法（LRFD）檢核上述柱構件之強度設計。（25 分）

**已知條件整理：**

| 參數 | 數值 |
|------|------|
| 柱長 $L$ | 400 cm |
| 斷面型式 | 正方形箱型（hollow square）|
| 外側寬度 $b_o$ | 80 cm |
| 板厚 $t$ | 3.2 cm |
| 設計壓力 $P_u$ | 1,119 tf |
| 有效長度係數 $K_1$ | 2.284（第一方向）|
| 有效長度係數 $K_2$ | 2.397（第二方向，控制）|
| 降伏強度 $F_y$ | 3.3 tf/cm² |
| 揚氏係數 $E$ | 2,040 tf/cm² |

**考卷給定參考公式：**

$$\lambda_c = \frac{KL}{\pi r}\sqrt{\frac{F_y}{E}}$$

$$\lambda_c > 1.5 \Rightarrow F_{cr} = \frac{0.877}{\lambda_c^2}F_y \quad (\text{彈性挫屈})$$

$$\lambda_c \leq 1.5 \Rightarrow F_{cr} = e^{-0.419\lambda_c^2}F_y \quad (\text{非彈性挫屈})$$

*（說明：$e^{-0.419\lambda_c^2} = 0.658^{\lambda_c^2}$，兩式等價）*

## 核心考點

**核心觀念：** 箱型斷面屬高效率斷面，對稱軸兩方向慣性矩相同，計算迴轉半徑時可利用代數化簡公式 $r^2 = (b_o^2 + b_i^2)/12$（僅適用正方形箱）。有效長度係數不同時，需分別計算兩方向之 $\lambda_c$ 取最大者控制。

**出題者意圖：**
1. 考核正方形箱型斷面斷面性質的計算（$A$、$I$、$r$）
2. 考核 LRFD 壓力桿件設計流程（$\lambda_c \to F_{cr} \to \phi_c P_n$）
3. 考核考生在兩個方向中能否正確選取控制方向

## 解題關鍵步驟

**作戰計畫：**
1. 計算箱型斷面面積 $A$ 與迴轉半徑 $r$
2. 計算兩方向之 $KL/r$，取最大者
3. 計算 $\lambda_c$，判斷挫屈類型
4. 計算 $F_{cr}$，再求 $\phi_c P_n = 0.85 F_{cr} A$
5. 比較 $P_u$ 與 $\phi_c P_n$，判定 OK / NG

**關鍵陷阱：**

| 陷阱 | 說明 | 應對 |
|------|------|------|
| ❌ 箱型斷面 $I$ 計算錯誤 | 容易忘記空心部分，或計算外形 $b^4/12$ 不扣內孔 | 用公式 $I = (b_o^4 - b_i^4)/12$，或利用化簡 $r^2 = (b_o^2+b_i^2)/12$ |
| ❌ 只計算一個方向 | 題目明給兩組 $K$ 值，需取控制方向（較大 $\lambda_c$）| 兩方向都算，取最大 $KL/r$ |
| ❌ 公式 $0.658^{\lambda_c^2}$ vs $e^{-0.419\lambda_c^2}$ 混淆 | 考卷給的是指數形式，計算時用計算機直接代入即可 | 兩式等價，擇一使用即可 |
| ❌ $\phi_c = 0.90$（誤） | LRFD 壓力桿件 $\phi_c = 0.85$，非 0.90 | 壓力用 0.85，彎曲用 0.90 |

## 4. 步驟化詳細計算過程 (Step-by-Step Detailed Calculation)

### Step 1：斷面幾何

**外側寬度：** $b_o = 80 \text{ cm}$

**內側寬度：** $b_i = b_o - 2t = 80 - 2(3.2) = 73.6 \text{ cm}$

**斷面積：**
$$A = b_o^2 - b_i^2 = 80^2 - 73.6^2 = 6400 - 5416.96 = \mathbf{983.04 \text{ cm}^2}$$

**斷面慣性矩（主軸，正方形對稱）：**
$$I = \frac{b_o^4 - b_i^4}{12}$$

**迴轉半徑（利用代數化簡）：**
$$r^2 = \frac{I}{A} = \frac{(b_o^4 - b_i^4)/12}{(b_o^2 - b_i^2)} = \frac{(b_o^2 + b_i^2)(b_o^2 - b_i^2)}{12(b_o^2 - b_i^2)} = \frac{b_o^2 + b_i^2}{12}$$

$$r^2 = \frac{80^2 + 73.6^2}{12} = \frac{6400 + 5416.96}{12} = \frac{11816.96}{12} = 984.75 \text{ cm}^2$$

$$\boxed{r = \sqrt{984.75} = 31.38 \text{ cm}}$$
