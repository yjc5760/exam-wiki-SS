## 題幹摘要

一熱軋 H 型鋼梁 **RH700×300×13×24**（$d \times b_f \times t_w \times t_f$，單位 mm），相關已知條件如下：

- 強軸慣性矩：$I_x = 197{,}000$ cm⁴
- 無側向支撐長度：$L_b = 3.0$ m
- 降伏強度：$F_y = 3.5$ tf/cm²
- 殘留應力（熱軋斷面）：$F_r = 0.7$ tf/cm²

試依 LRFD 規範：
1. 計算 $M_p$（塑性彎矩）
2. 計算 $M_r$（彈性限制彎矩）
3. 確認 LTB 控制區間，求設計撓曲強度 $\phi_b M_n$

## 核心考點

**核心觀念：LRFD 梁的三段式 LTB 設計流程**

本題考查完整的梁設計流程：
1. 計算 $Z_x$（需要理解塑性中性軸和塑性斷面模數的幾何意義）
2. 計算 $M_p$ 和 $M_r$（確立 LTB 曲線的上下界）
3. 比較 $L_b$ 與 $L_p$，確定 LTB 控制類型
4. 最終得出 $M_n$ 與 $\phi_b M_n$

**出題者給出 $M_r$ 的用意：** 要求學生知道 $M_r = (F_y - F_r)S_x$（殘留應力使彈性限制彎矩降低），理解殘留應力在 LTB 曲線中的角色——$M_r$ 是 LTB 非彈性段和彈性段的分界點（在 $L_r$ 處的 $M_n$）。

**出題者給出 $F_r = 0.7$ tf/cm²（非 1.16 tf/cm²）的暗示：** 熱軋斷面，殘留應力較銲接組合斷面小。

## 解題關鍵步驟

**步驟規劃：**
1. 由斷面標示推算幾何尺寸 → 計算 $Z_x$、$S_x$
2. 計算 $M_p = F_y Z_x$
3. 計算 $M_r = (F_y - F_r) S_x$
4. 計算 $L_p$（或用題目給定值）並與 $L_b$ 比較
5. 確認 $L_b < L_p$ → $M_n = M_p$ → 計算 $\phi_b M_n$

**關鍵陷阱：**

> ⚠️ **陷阱1：$S_x$ 的計算基準**
> $S_x = I_x / (d/2)$，其中 $d$ 是**全深**（包含兩翼板厚）= 70 cm。故 $S_x = 197{,}000 / 35 = 5{,}628.6$ cm³，不是用 $h_w/2 = 32.6$ cm。

> ⚠️ **陷阱2：$Z_x$ vs $S_x$ 的計算式**
> $Z_x$ 的翼板形心距 = $h_w/2 + t_f/2$（到塑性中性軸），而不是 $d/2$（那是彈性計算中 $S_x$ 的基準）。

> ⚠️ **陷阱3：$F_r$ 的角色**
> $F_r$ 不直接影響 $M_p$（$M_p = F_y Z_x$ 與殘留應力無關），只影響 $M_r = (F_y - F_r)S_x$（彈性範圍受殘留應力限制）。

> ⚠️ **陷阱4：$M_r$ 的應用場合**
> 本題 $L_b < L_p$，最終 $M_n = M_p$，$M_r$ 本身不直接代入 $M_n$ 公式，但仍需計算作為 LTB 線性插值公式的下界（萬一 $L_b$ 介於 $L_p$ 和 $L_r$ 之間時才用到插值：$M_n = M_p - (M_p - M_r)(L_b - L_p)/(L_r - L_p)$）。
