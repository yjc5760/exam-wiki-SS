## 計算題：五大主題識別

| 題目給的關鍵資訊 | 主題 |
|----------------|------|
| 截面 + KL 或 KL/r | 壓力桿件 |
| 截面 + Lb + 彎矩 | 梁桿件（LTB）|
| 截面 + KL + 彎矩（同時有） | 梁柱桿件 |
| 螺栓群 / 銲縫群 | 接合設計 |
| 結構 + 問極限載重 | 塑性分析 |

## 壓力桿件流程

```
① 確認兩軸 KL/r，取 max         ← 陷阱：兩軸都要查，K 值要對
② 算 λc = (KL/rπ)√(Fy/E)
   λc ≤ 1.5 → Fcr = 0.658^(λc²) × Fy
   λc > 1.5 → Fcr = (0.877/λc²) × Fy
③ LRFD：φcPn = 0.85 × Fcr × Ag
   ASD：Fa = Fcr/FS（FS 代公式，非固定 12/5）
```

---

## 梁桿件流程（LTB）

```
① 弱軸？→ 無 LTB，Mn = min(Mp, 1.5FySy)，結束
② 強軸 → 算 Cb（彎矩梯度係數）
   ← 陷阱：懸臂梁 Cb = 1.0，不套公式
③ 算 Lp、Lr，對照 Lb：
   Lb ≤ Lp        → Mn = Mp = FyZx
   Lp < Lb ≤ Lr   → Mn = Cb[Mp-(Mp-0.7FySx)(Lb-Lp)/(Lr-Lp)] ≤ Mp
   Lb > Lr         → Mn = FcrSx ≤ Mp
   ← 陷阱：乘 Cb 後必須截頂 Mp
④ 剪力：φvVn = 1.0 × 0.6Fy × Aw（一般 H 型鋼）
   ← 陷阱：Zx ≠ Ix/c（塑性 ≠ 彈性斷面模數）
```

---

## 梁柱桿件流程（P-M 互制）

```
① 分別算 φcPn（壓力流程）和 φbMnx（梁流程）
② 需要二階放大？
   Mr = B1·Mnt + B2·Mlt
   B1 = Cm/(1-Pu/Pe1)   Pe1 用不側移 K（K ≤ 1）
   B2 = 1/(1-ΣPu/ΣPeK)  PeK 用側移 K（K > 1），整層所有柱加總
   ← 陷阱：B2 漏算重力柱；Pe1 和 PeK 的 K 不同
③ 算 Pr/Pc 判斷互制式版本：
   Pr/Pc ≥ 0.2 → Pr/Pc + 8/9(Mrx/Mcx + Mry/Mcy) ≤ 1.0
   Pr/Pc < 0.2 → Pr/(2Pc) + (Mrx/Mcx + Mry/Mcy) ≤ 1.0
```

## 接合設計流程

```
【螺栓】
① 單剪 or 雙剪？承壓型 or 摩阻型？
② 計算各破壞模式，取最小：
   剪力：φFnvAb
   承壓：φ2.4Fudt
   淨截面：φ0.75FuAe（Ae = UAn）
   塊狀剪力：φ[0.6FuAnv + UbsFuAnt]
   ← 陷阱：Ubs（均勻拉力=1.0；非均勻=0.5）；拉力面用 Fu 不是 Fy
③ 摩阻型不可與銲縫共用承力（新設接合）

【銲縫（偏心）】
① 找銲縫群形心位置
② 算 Ip（單位 cm³，不是 cm⁴）
③ 找最遠受力點，向量合成直接力+扭矩力
```

## 塑性分析流程

```
① 靜不定度 n → 崩塌所需塑性鉸數 = n + 1
   ← 陷阱：塑性鉸仍承受 Mp，不是 M = 0
② 列所有可能崩塌機構（位置必須能形成幾何可動機構）
③ 虛功法：Σ(外力×位移) = Σ(Mp×θ)
   不等跨：θ₁ = δ/x，θ₂ = δ/(L-x)，中間鉸轉角 = θ₁ + θ₂
④ 各機構取最小崩塌載重
```

## 說明題：固定題庫（7 主題）

| 主題 | 答題架構 |
|------|---------|
| BH vs RH | 製造方式 → 殘留應力 → 斷面彈性 |
| NDT 選擇 | 表面缺陷（MT/PT）→ 內部裂縫（UT）→ 內部氣孔（RT）→ CJP 要 VT+UT |
| RBS 犬骨接頭 | 削翼板非腹板 → 控制塑性鉸遠離銲接區 → 容量設計邏輯 |
| SN 鋼材 | 背景（阪神地震）→ 特色（上下限+降伏比≤0.80+CVN）→ C 級 Z 向韌性 |
| 強柱弱梁 | 目的 → 驗核式 ΣMpc≥ΣMpb → Ry 期望強度 |
| 防蝕三類 | 塗裝（隔絕）→ 鍍鋅（隔絕+犧牲陽極）→ 陰極保護（電化學）|
| 容量設計 | 識別脆弱點 → 強化脆弱點 → 弱化控制點 → Ry 工具 |

## 關鍵輸入值速記

| 輸入值 | 最常見陷阱 |
|--------|---------|
| Cb | 懸臂梁 = 1.0；跨中集中力 > 均布 |
| K | 側移 vs 不側移，B1 和 B2 用不同 K |
| Ae = UAn | 銲接接合 An = Ag，但 U 不同 |
| Zx | 塑性斷面模數，≠ Ix/c |
| Ip | 單位 cm³ |
| Ubs | 均勻拉力=1.0，非均勻=0.5 |
| ΣPeK | 整層所有柱，重力柱也算 |
