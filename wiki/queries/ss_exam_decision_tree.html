壓力桿件流程

  • 1
    確認兩軸有效細長比，取較大值
    (KL/r)_x vs (KL/r)_y→ 取 max
    ⚠ 陷阱 兩軸必須都查，K 值依支撐條件選對位圖版本
  • 2
    算 λc 判斷彈性/非彈性
    λc = (KL/rπ)√(Fy/E)
    λc ≤ 1.5：Fcr = 0.658^(λc²) × Fy（非彈性）
    λc > 1.5：Fcr = (0.877/λc²) × Fy（彈性）
  • 3
    算設計強度
    LRFD：φcPn = 0.85 × Fcr × Ag
    ASD：Fa = Fcr/FS，FS = 5/3 + 3(KL/r)/8Cc − (KL/r)³/8Cc³
    ⚠ 陷阱 ASD 的 FS 不是固定 12/5，要代公式

梁桿件流程（LTB）

  • 1
    判斷強軸 or 弱軸彎曲
    弱軸 → ✓ 無 LTB 直接 Mn = min(Mp, 1.5FySy)，結束
    強軸 → 繼續往下
  • 2
    算 Cb（彎矩梯度係數）
    Cb = 12.5Mmax / (2.5Mmax+3MA+4MB+3MC)
    均布載重下，Cb ≈ 1.14；集中載重下，Cb ≈ 1.32
    ⚠ 陷阱 懸臂梁 Cb = 1.0（不得用公式）
  • 3
    算 Lp、Lr，對照 Lb 判斷區段
    Lb ≤ Lp：Mn = Mp = FyZx ✓ 塑性
    Lp<Lb≤Lr：Mn = Cb[Mp−(Mp−0.7FySx)(Lb−Lp)/(Lr−Lp)] ≤ Mp
    Lb > Lr：Mn = FcrSx ≤ Mp（彈性 LTB）
    ⚠ 陷阱 乘 Cb 後不可超過 Mp，要截頂
  • 4
    剪力強度（若題目有問）
    φvVn = 1.0 × 0.6Fy × Aw（腹板 h/tw ≤ 2.24√(E/Fy) 時）
    ⚠ 陷阱 Zx ≠ Ix/c，兩者公式完全不同

梁柱桿件流程（P-M 互制）

  • 1
    分別算 φcPn 和 φbMnx
    壓力：走「壓力桿件流程」
    彎矩：走「梁桿件流程」
  • 2
    需要二階放大？算 Mr = B1Mnt + B2Mlt
    B1 = Cm/(1−Pu/Pe1)，Pe1 用不側移 K
    B2 = 1/(1−ΣPu/ΣPeK)，PeK 用側移 K，對整層所有柱加總
    ⚠ 陷阱 B2 漏算重力柱；Pe1 和 PeK 的 K 值不同
  • 3
    判斷互制公式版本（先算 Pr/Pc）
    Pr/Pc ≥ 0.2：Pr/Pc + 8/9(Mrx/Mcx + Mry/Mcy) ≤ 1.0
    Pr/Pc < 0.2：Pr/(2Pc) + (Mrx/Mcx + Mry/Mcy) ≤ 1.0

接合設計流程

  • 1
    識別接合類型
    螺栓群：承壓型 or 摩阻型？單剪 or 雙剪？
    銲縫群：填角銲 or 開槽銲？同心 or 偏心？
  • 2
    螺栓：計算各破壞模式，取最小
    剪力：φRn = φFnvAb
    承壓：φRn = φ2.4Fudt
    摩阻型另加：φRn = φμDuhfTbns
    淨截面：φ0.75FuAe（Ae = UAn）
    塊狀剪力：φ[0.6FuAnv + UbsFuAnt]
    ⚠ 陷阱 摩阻型螺栓不與銲縫共用（新設接合）
  • 3
    偏心銲縫：Ip 法
    1. 找形心位置
    2. 算 Ip（單位 cm³，非 cm⁴）
    3. 找最遠受力點，向量合成
    ⚠ 陷阱 Ip 單位是 cm³，面積矩（長度×長度²）
  • 4
    塊狀剪力的拉力面判斷
    均勻拉力（多行螺栓）：Ubs = 1.0
    非均勻（如 L 型角材）：Ubs = 0.5
    ⚠ 陷阱 拉力面永遠用 Fu，不是 Fy

塑性分析流程

  • 1
    確認結構條件
    靜不定度 n → 崩塌所需塑性鉸數 = n + 1
    斷面必須為結實斷面（λ ≤ λp）
    ⚠ 陷阱 塑性鉸仍承受 Mp，不是 M = 0
  • 2
    列出所有可能崩塌機構
    每種機構畫出塑性鉸位置，確認能形成幾何可動機構
  • 3
    對每個機構用虛功法
    Σ(外力 × 位移) = Σ(Mp × θ)
    不等跨：θ₁ = δ/x，θ₂ = δ/(L−x)，中間鉸轉角 = θ₁ + θ₂
    ⚠ 陷阱 中間鉸的轉角是兩段轉角之和
  • 4
    取各機構的最小崩塌載重
    Wc = min(各機構解) → 這就是極限載重

點選主題查看考官版答題架構

BH vs RH 差異（答題三段）
  • 製造方式：BH = 三塊板填角銲焊接，RH = 熱軋一體成形
  • 殘留應力：BH 翼腹交接處拉應力；RH 翼板端部壓應力 ≈ 0.3Fy
  • 斷面彈性：BH 可客製大尺寸；RH 受限標準規格
NDT 方法選擇邏輯
  • 表面缺陷：MT（磁性材料）或 PT（所有非多孔材料）
  • 內部面積型（裂縫）：UT 超音波（裂縫面是良好反射面）
  • 內部體積型（氣孔、夾渣）：RT 射線
  • CJP 完全滲透銲縫：VT + UT（或 RT）必須同時做
RBS 犬骨接頭（容量設計邏輯）
  • 削弱的是翼板，不是腹板（距柱面 a 到 a+b 區間）
  • 目的：控制塑性鉸位置遠離脆弱銲接區
  • 邏輯：削弱梁的 Mp → 接頭所受力也降低 → 保護接頭
  • 記憶：「弱化可控位置，保護脆弱位置」
SN 鋼材（背景→特色→數值三段）
  • 背景：1995 阪神地震，傳統鋼材只有 Fy 下限 → 實際超強 → 容量設計失效
  • 特色：Fy 上下限雙重控制、降伏比 ≤ 0.80、CVN ≥ 27J @ 0°C
  • C 級額外：Z 向韌性 RA ≥ 25%（防層狀撕裂，用於厚板柱）
強柱弱梁驗核流程
  • 目的：讓梁先降伏，柱維持彈性，保持側向穩定
  • 驗核式：ΣMpc ≥ ΣMpb（用期望強度 RyFyZ）
  • Ry：A992 = 1.1，A36 = 1.5，SN490B = 1.1
  • 記憶：強柱 = 柱的塑性矩加總 ≥ 梁的塑性矩加總
防蝕三大類（需說明機制）
  • 塗裝：物理隔絕（阻擋氧氣和水分）
  • 金屬鍍層（熱浸鍍鋅）：物理隔絕＋犧牲陽極（鋅先腐蝕保護鋼）
  • 電化學保護（陰極保護）：外加電流或犧牲陽極，使鋼材成為陰極
容量設計哲學
  • 核心：讓可塑性構件先降伏，保護脆弱接合部
  • 工具：Ry 超強係數（期望強度 = RyFyZ）
  • 應用：強柱弱梁、RBS 接頭、Panel Zone 設計
  • 邏輯順序：識別脆弱點 → 強化脆弱點 → 弱化控制點
